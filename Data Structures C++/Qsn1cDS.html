Q1. Write a C++ program that uses functions to perform the following:
c) Display the contents of the above list after deletion.

Aim: Display the contents of the above list after deletion.

System and Software Requirements: A Computer with any Operating system with G++ compiler installed in it.

Description:
we have a linked list that consists of nodes. Each node contains the data and address to the next node. After inserting the data in the linked list, we will create a function to delete the first node. we will create a temporary pointer that initially points to the head and move the head to the next node. Now delete the temporary node and return the linked list.

Algorithm:
Step-1: A function takes the pointer to the head and deletes the first node of the linked list.
Step-2: Create a temporary pointer that initially points to the head.
Step-3: Head moves to the next node.
Step-4:Delete the temporary pointer.
Step-5:Return the linked list.

/* Link list node */
struct Node {

/* Function to remove the first node

Node* removeFirstNode(struct Node* head)
{
if (head == NULL)
return NULL;
// Move the head pointer to the next node
Node* temp = head;
head = head->next;
delete temp;
return head;
}

void push(struct Node** head_ref, int new_data)
{
struct Node* new_node = new Node;
new_node->data = new_data;
new_node->next = (*head_ref);
(*head_ref) = new_node;
}
