Q17. Write a C++ program to implement extendible hashing.

Aim: A C++ program to implement extendible hashing.

System and Software Requirements: A Computer with any Operating system with G++ compiler installed in it.

Description:
Extendible hashing is a hash-based indexing technique that is commonly used in database systems to efficiently store and retrieve large amounts of data. In this technique, the hash function maps a key to a bucket, which contains one or more records. However, if the number of records in a bucket exceeds a certain threshold, the bucket must be split into two, and the hash function must be updated to map keys to the appropriate new bucket.
Algorithm:
To implement extendible hashing in a C++ program, we can follow the following steps:
Step-1: Define a class to represent a bucket. The bucket should have a data structure to store the records, and a function to check if it is full.
Step-2: Define a class to represent the hash table. The hash table should have a fixed number of buckets, and a function to compute the hash value of a key.
Step-3: When a record needs to be inserted, compute its hash value and find the corresponding bucket in the hash table. If the bucket is not full, insert the record into it. Otherwise, split the bucket into two and update the hash function to map keys to the appropriate new bucket.
Step-4: When a record needs to be retrieved, compute its hash value and find the corresponding bucket in the hash table. Search the bucket for the record. If it is found, return it. Otherwise, the record does not exist in the hash table.
Step-5: When a record needs to be deleted, compute its hash value and find the corresponding bucket in the hash table. Search the bucket for the record. If it is found, remove it. If the bucket is empty after the removal, merge it with a neighboring bucket if possible.
To implement extendible hashing efficiently, we can use techniques such as caching and lazy splitting. Caching involves keeping frequently accessed buckets in memory, while lazy splitting involves delaying the splitting of a bucket until it is absolutely necessary.
In conclusion, implementing extendible hashing in a C++ program requires defining classes for buckets and the hash table, and implementing functions to insert, retrieve, and delete records. To improve performance, caching and lazy splitting techniques can be used.

#include<iostream>
#include<vector>
#include<string>
using namespace std;
// Class for a bucket
class Bucket {
public:
vector data;
int localDepth;
};
// Class for a directory
class Directory {
public:
vector buckets;
int globalDepth;
};
// Class for a hash table
class HashTable {
public:
int bucketSize;
vector buckets;
Directory directory;
};
// Function to split a bucket
void splitBucket(int bucketIndex, HashTable& hashTable) {
int localDepth = hashTable.buckets[bucketIndex].localDepth;
int newBucketIndex = bucketIndex + (1 << (localDepth-1));
Bucket newBucket;
newBucket.localDepth = localDepth + 1;
Bucket& oldBucket = hashTable.buckets[bucketIndex];
for (int i = oldBucket.data.size()-1; i >= 0; i--) {
int hash = (oldBucket.data[i] >> (localDepth-1)) & 1;
if (hash == 1) {
newBucket.data.push_back(oldBucket.data[i]);
oldBucket.data.erase(oldBucket.data.begin() + i);
}
}
hashTable.buckets[newBucketIndex] = newBucket;
hashTable.buckets[bucketIndex].localDepth++;
}
// Function to update the directory
void updateDirectory(int bucketIndex, HashTable& hashTable) {
int localDepth = hashTable.buckets[bucketIndex].localDepth;
int globalDepth = hashTable.directory.globalDepth;
if (localDepth == globalDepth) {
hashTable.directory.buckets.resize(1 << globalDepth);
for (int i = 0; i < (1 << globalDepth); i++) {
hashTable.directory.buckets[i] = i & ((1 << localDepth) - 1);
}
hashTable.directory.globalDepth++;
} else {
int splitIndex = bucketIndex & ((1 << localDepth) - 1);
int baseIndex = bucketIndex - splitIndex;
int numBuckets = 1 << (globalDepth - localDepth);
for (int i = splitIndex; i < numBuckets; i += (1 << localDepth)) {
hashTable.directory.buckets[baseIndex + i] = bucketIndex;
}
}
}
// Function to insert data into the hash table
void insert(int data, HashTable& hashTable) {
int hash = data & ((1 << hashTable.directory.globalDepth) - 1);
int bucketIndex = hashTable.directory.buckets[hash];
Bucket& bucket = hashTable.buckets[bucketIndex];
if (bucket.data.size() == hashTable.bucketSize) {
splitBucket(bucketIndex, hashTable);
updateDirectory(bucketIndex, hashTable);
insert(data, hashTable);
} else {
bucket.data.push_back(data);
}
}

HashTable hashTable;
hashTable.bucketSize = 2;

hashTable.directory.globalDepth = 1;
hashTable.directory.buckets.resize(2);
