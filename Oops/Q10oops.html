Q10. Write a Java program that loads names and phone numbers from a text file where the data is organized as one line per record and each field in a record are separated by a tab (\t). It takes a name or phone number as input and prints the corresponding other value from the hash table(hint: use hash tables).

Aim: To develop a Java program that loads data from a text file, where each record is separated by a tab (\t), and uses a hash table to efficiently search for names and phone numbers. The program should take input from the user in the form of a name or phone number and output the corresponding value from the hash table.

System and Software Requirements: A Computer with any Operating system with javac compiler installed in it.

Description:
The problem statement requires a Java program to read a text file containing names and phone numbers, where each record is separated by a tab character. The program needs to create a hash table to store the data, with the name and phone number as key-value pairs. The program should take either a name or phone number as input and retrieve the corresponding other value from the hash table.
To solve this problem, the following steps can be followed:
1. Create a hash table object in Java. A hash table is a data structure that allows key-value pairs to be stored and retrieved efficiently.
2. Read the text file line by line using a file input stream. For each line, split the line into two parts using the tab character as the delimiter. The first part will be the name and the second part will be the phone number.
3. Add the name and phone number to the hash table as a key-value pair. The name should be the key and the phone number should be the value.
4. Once all the records have been read and added to the hash table, prompt the user to input either a name or a phone number.
5. If the user inputs a name, retrieve the corresponding phone number from the hash table by using the name as the key. If the user inputs a phone number, retrieve the corresponding name from the hash table by using the phone number as the value.
6. Print the corresponding value retrieved from the hash table to the console.
In summary, the Java program needs to read a text file containing names and phone numbers, create a hash table to store the data, and retrieve the corresponding value based on user input. This can be achieved by using a file input stream to read the text file, a hash table to store the data, and using the key-value pairs in the hash table to retrieve the corresponding value based on user input.

Suppose we have a text file named "contacts.txt" containing the following data:

import java.io.BufferedReader;
import java.io.FileReader;
import java.io.IOException;
import java.util.HashMap;
import java.util.Map;
public class ContactFinder {
private static final String FILENAME = "contacts.txt";
private static final String SEPARATOR = "\t";
public static void main(String[] args) {
Map contacts = loadContacts();
String searchValue = "John"; // replace with user input
String result = contacts.get(searchValue);
if (result != null) {
System.out.println("The phone number of " + searchValue + " is " + result);
} else {
searchValue = "555-1234"; // replace with user input
for (Map.Entry entry : contacts.entrySet()) {
if (entry.getValue().equals(searchValue)) {
System.out.println("The name of " + searchValue + " is " + entry.getKey());
break;
}
}
}
}
private static Map loadContacts() {
Map contacts = new HashMap<>();
try (BufferedReader br = new BufferedReader(new FileReader(FILENAME))) {
String line;
while ((line = br.readLine()) != null) {
String[] parts = line.split(SEPARATOR);
contacts.put(parts[0], parts[1]);
}
} catch (IOException e) {
e.printStackTrace();
}
return contacts;
}
}
