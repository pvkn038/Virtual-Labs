Q14. Write a Java program that implements Bubble sort algorithm for sorting in descending order and also show the number of interchanges occurred for the given set of integers.

Aim: To implements the Bubble sort algorithm for sorting a given set of integers in descending order and also displays the number of interchanges that occurred during the sorting process.

System and Software Requirements: A Computer with any Operating system with javac compiler installed in it.

Description:
Bubble sort is a simple sorting algorithm that repeatedly steps through the list, compares adjacent elements and swaps them if they are in the wrong order. The algorithm gets its name from the way smaller elements "bubble" to the top of the list.
To implement the Bubble sort algorithm in Java for sorting in descending order, we can modify the comparison condition to check if the current element is less than the next element. If so, we swap them. This will result in the larger elements "bubbling" to the top of the list.
The basic algorithm for Bubble sort in descending order is as follows:
1. Start at the first element of the list.
2. Compare the current element with the next element.
3. If the current element is less than the next element, swap them.
4. Move to the next pair of adjacent elements and repeat steps 2 and 3 until the end of the list is reached.
5. If any swaps were made in the previous pass, repeat steps 1-4 again.
6. Stop when no swaps are made in a pass.
To count the number of interchanges occurred during the sorting process, we can increment a counter each time we swap two elements.
The number of interchanges for a set of n elements in the worst-case scenario for Bubble sort is (n-1) + (n-2) + ... + 1, which is equal to n*(n-1)/2. This means that the number of interchanges is proportional to the square of the number of elements being sorted.
In summary, implementing Bubble sort in Java for descending order involves modifying the comparison condition and counting the number of interchanges that occur during the sorting process. The number of interchanges is proportional to the square of the number of elements being sorted.



Examples:
Suppose we have the following set of integers that we want to sort in descending order using Bubble sort: [5, 2, 9, 1, 7]
1. Start at the first element of the list, which is 5.
2. Compare 5 with the next element, which is 2.
3. 5 is greater than 2, so we don't swap them.
4. Move to the next pair of adjacent elements, which are 5 and 9.
5. 5 is less than 9, so we swap them. The list becomes [9, 2, 5, 1, 7] and we increment the interchange counter to 1.
6. Move to the next pair of adjacent elements, which are 9 and 1.
7. 9 is greater than 1, so we swap them. The list becomes [9, 2, 1, 5, 7] and we increment the interchange counter to 2.
8. Move to the next pair of adjacent elements, which are 9 and 5.
9. 9 is greater than 5, so we swap them. The list becomes [9, 5, 1, 2, 7] and we increment the interchange counter to 3.
10. Move to the next pair of adjacent elements, which are 9 and 7.
11. 9 is greater than 7, so we swap them. The list becomes [9, 7, 1, 2, 5] and we increment the interchange counter to 4.
12. We've reached the end of the list, but we made swaps in the previous pass, so we repeat steps 1-11.
13. Start at the first element of the list, which is 9.
14. Compare 9 with the next element, which is 7.
15. 9 is greater than 7, so we swap them. The list becomes [7, 9, 1, 2, 5] and we increment the interchange counter to 5.
16. Move to the next pair of adjacent elements, which are 9 and 1.
17. 9 is greater than 1, so we swap them. The list becomes [7, 9, 2, 1, 5] and we increment the interchange counter to 6.
18. Move to the next pair of adjacent elements, which are 9 and 2.
19. 9 is greater than 2, so we swap them. The list becomes [7, 9, 2, 1, 5] and we increment the interchange counter to 7.
20. Move to the next pair of adjacent elements, which are 9 and 5.
21. 9 is greater than 5, so we swap them. The list becomes [7, 9, 5, 1, 2] and we increment the interchange counter to 8.
22. We've reached the end of the list and no swaps were made in the previous pass, so we stop.
23. The sorted list is [7, 9, 5, 2, 1] and the number of interchanges is 8.

import java.util.Scanner;
public class BubbleSort {
public static void main(String[] args) {
Scanner read = new Scanner(System.in);
int size, count = 0;
//Reading size of the list
System.out.print("Enter the list size: ");
size = read.nextInt();
//Creating list with elements
int list[] = new int[size];
System.out.println("Enter any " + size + " integer numbers: ");
for(int i = 0; i < size; i++)
list[i] = read.nextInt();
// Bubble sort logic
int temp=0;
for(int i=0;i<size-1;i++) {
for(int j=0;j<size-i-1;j++) {
if(list[j]<list[j+1]) {
temp=list[j];
list[j]=list[j+1];
list[j+1]=temp;
count++;
}
}
}
// Displaying sorted list
System.out.println("List of sorted elements: ");
for(int x:list) {
System.out.print(x + " ");
}
System.out.println("\nTotal number of Interchanges is " + count);
}
}

import java.util.Scanner;
public class BubbleSortDescending {
public static void main(String[] args) {
Scanner input = new Scanner(System.in);
// Get the number of elements to sort
System.out.print("Enter the number of elements: ");
int n = input.nextInt();
// Create an array to hold the elements
int[] arr = new int[n];
// Get the elements from the user
System.out.print("Enter the elements: ");
for (int i = 0; i < n; i++) {
arr[i] = input.nextInt();
}
// Sort the elements in descending order using Bubble sort
int numSwaps = 0;
for (int i = 0; i < n - 1; i++) {
for (int j = 0; j < n - i - 1; j++) {
if (arr[j] < arr[j+1]) {
// Swap the elements
int temp = arr[j];
arr[j] = arr[j+1];
arr[j+1] = temp;
numSwaps++;
}
}
}
// Display the sorted elements and the number of swaps
System.out.print("Sorted array in descending order: ");
for (int i = 0; i < n; i++) {
System.out.print(arr[i] + " ");
}
System.out.println("\nNumber of swaps: " + numSwaps);
}
}
