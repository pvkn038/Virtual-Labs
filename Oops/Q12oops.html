Q12. Write a Java program to list all the files in a directory including the files present in all its Sub directories.

Aim: To develop a Java program that can recursively list all the files present in a directory and its subdirectories.

System and Software Requirements: A Computer with any Operating system with javac compiler installed in it.

Description:
The experiment theory to list all the files in a directory including the files present in all its subdirectories in Java can be formulated as follows:
1. The experiment will be conducted on a computer system with Java installed, and a directory containing files and subdirectories will be chosen as the target directory for the program.
2. The Java program will use the File class to access the target directory and its subdirectories.
3. The program will traverse the directory tree recursively, starting from the target directory, to list all files present in it, including those in its subdirectories.
4. The program will use a recursive function to iterate over each subdirectory and its files, until all files in the target directory and its subdirectories have been listed.
5. The program will use the listFiles() method of the File class to obtain a list of all files and directories in a given directory.
6. The program will use the isDirectory() method of the File class to identify subdirectories and recursively call the same function on each subdirectory.
7. The program will use a loop to iterate over each file in a directory and print its name to the console.
8. The experiment will be repeated with different target directories and subdirectories to verify the program's effectiveness and to ensure that it works correctly for all directories and files.
9. Finally, the program's performance will be evaluated by measuring the time it takes to list all files in a large directory tree and comparing it with the time taken by other similar programs or methods.



Examples:
Let's say we have a directory called "my_directory" located at "C:\Users\Username\Documents\my_directory" on a Windows machine. This directory contains several files and subdirectories.
The goal of our Java program is to list all the files in this directory, including the files in its subdirectories.
To accomplish this, we will use a recursive function that starts at the "my_directory" level, checks whether each item in the directory is a file or a subdirectory, and then repeats the process for each subdirectory until all files have been listed.
For example, the program might follow these steps:
1. Start at "C:\Users\Username\Documents\my_directory".
2. Check whether each item in the directory is a file or a subdirectory.
3. If the item is a file, print its name to the console.
4. If the item is a subdirectory, call the same function on that subdirectory, repeating steps 2-4 until all files have been listed.
Assuming the "my_directory" contains a file called "file1.txt" and a subdirectory called "subdir1" which contains a file called "file2.txt", the output of our program might look like this:
file1.txt
file2.txt
Note that this is a simplified example for illustration purposes only. In practice, the program would need to handle more complex directory structures and file types.

public static void printFiles(String path) {
System.out.println("Current Location: " + path);
File f_ref = new File(path);
File[] filesList = f_ref.listFiles();
for (File file : filesList) {
if (file.isFile())
System.out.println("- " + file.getName());
else

import java.io.File;
public class ListFiles {
public static void main(String[] args) {
// Directory path
String path = "C:/Users/UserName/Desktop/Folder";
// Create a File object for the directory
File directory = new File(path);
// Get all files in the directory and its subdirectories
File[] files = directory.listFiles();
// Iterate through all files and print their names
for (File file : files) {
if (file.isFile()) {
System.out.println(file.getName());
} else if (file.isDirectory()) {
listFiles(file); // Recursive call
}
}
}
// Helper method for recursive directory traversal
public static void listFiles(File directory) {
File[] files = directory.listFiles();
for (File file : files) {
if (file.isFile()) {
System.out.println(file.getName());
} else if (file.isDirectory()) {
listFiles(file); // Recursive call
}
}
}
