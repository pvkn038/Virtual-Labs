Q13. Write a Java program that implements Quick sort algorithm for sorting a list of names in ascending order.

Aim: To implement the Quick sort algorithm for sorting a list of names in ascending order. The program will take an unsorted list of names as input and use the Quick sort algorithm to sort the list in ascending order. The program will then output the sorted list of names.

System and Software Requirements: A Computer with any Operating system with javac compiler installed in it.

Description:
Quick sort is a popular divide-and-conquer sorting algorithm used to sort arrays or lists. It works by dividing an array or list into two sub-arrays or sub-lists and recursively sorting each sub-array or sub-list.
To implement Quick sort algorithm for sorting a list of names in ascending order in Java, we can follow the following steps:
1. Define a method that takes the list of names as input parameter.
2. In the method, we first check if the list is empty or contains only one element. If it is, then the list is already sorted and we can return the list.
3. If the list has more than one element, we choose a pivot element from the list. The pivot element can be chosen randomly or as the first, last, or middle element of the list.
4. We partition the list into two sub-lists, such that all elements less than the pivot are in the left sub-list and all elements greater than the pivot are in the right sub-list. This is done by iterating through the list and swapping elements that are out of place.
5. After partitioning, the pivot element is in its correct position in the list. We then recursively sort the left and right sub-lists using the same algorithm until the sub-lists contain only one element or are empty.
6. Finally, we merge the sorted sub-lists to get the sorted list of names.
The worst-case time complexity of Quick sort is O(n^2), but on average, it has a time complexity of O(n log n) and is widely used due to its efficiency in practice.



Examples:
For example, consider the following unsorted array of names:
{"Emily", "Bob", "Alice", "David", "Catherine"}
The algorithm would first select a pivot, for instance, "Emily", and then partition the array around the pivot, such that all the names less than "Emily" are moved to the left of the pivot, and all the names greater than "Emily" are moved to the right of the pivot.
{"Bob", "Alice", "David", "Catherine", "Emily"}
The algorithm would then recursively apply the same process to the subarrays on either side of the pivot until the entire array is sorted. For instance, the algorithm would select "Bob" as the pivot for the left subarray, and partition it around the pivot, resulting in the following:
{"Alice", "Bob", "David", "Catherine", "Emily"}
This process would then be repeated for the left subarray until it is sorted, resulting in the following sorted array:
{"Alice", "Bob", "Catherine", "David", "Emily"}
Quick sort is a widely used algorithm for sorting elements in an array, and its efficiency and effectiveness make it a popular choice for many programming applications.

return;

this.names = array;

void quickSort(int lowerIndex, int higherIndex) {
int i = lowerIndex;
int j = higherIndex;
String pivot = this.names[lowerIndex + (higherIndex - lowerIndex) / 2];
while (i <= j) {
while (this.names[i].compareToIgnoreCase(pivot) < 0) {
i++;
}
while (this.names[j].compareToIgnoreCase(pivot) > 0) {
j--;

void exchangeNames(int i, int j) {

public static void quickSort(String[] arr, int low, int high) {
if (low < high) {
int pivotIndex = partition(arr, low, high);
quickSort(arr, low, pivotIndex - 1);
quickSort(arr, pivotIndex + 1, high);
}
}
public static int partition(String[] arr, int low, int high) {
String pivot = arr[high];
int i = low - 1;
for (int j = low; j <= high - 1; j++) {
if (arr[j].compareToIgnoreCase(pivot) < 0) {
i++;
String temp = arr[i];
arr[i] = arr[j];
arr[j] = temp;
}
}
String temp = arr[i + 1];
arr[i + 1] = arr[high];
arr[high] = temp;
return i + 1;
}
