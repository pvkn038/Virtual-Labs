Q9. Write a Java program that handles all mouse events and shows the event name at the center of the window when a mouse event is fired (Use Adapter classes).

Aim: To create a Java program that handles all mouse events using adapter classes and displays the event name at the center of the window when a mouse event is fired (Use Adapter classes).

System and Software Requirements: A Computer with any Operating system with javac compiler installed in it.

Description:
In Java, events generated by the mouse, such as clicking, dragging, or releasing a button, can be handled using the MouseEvent class and its associated listeners. However, instead of implementing all the methods of a listener interface, such as MouseListener, MouseMotionListener, or MouseWheelListener, we can use adapter classes that provide default implementations for all the methods of these interfaces.
The adapter classes for mouse events are MouseAdapter, MouseMotionAdapter, and MouseWheelAdapter. To handle all mouse events and show the event name at the center of the window, we can create a class that extends one of these adapter classes, depending on the types of mouse events we want to handle.
In the class, we can override the methods that correspond to the desired events, and inside each method, we can display the event name at the center of the window using the Graphics class and the drawString() method.
For example, if we want to handle all mouse events, we can create a class that extends MouseAdapter, and override the following methods:
mousePressed(MouseEvent e)
mouseReleased(MouseEvent e)
mouseClicked(MouseEvent e)
mouseEntered(MouseEvent e)
mouseExited(MouseEvent e)
In each method, we can get the event name using the getName() method of the MouseEvent class, and display it at the center of the window using the Graphics class and the drawString() method.
Finally, to use this class, we can create an instance of it and add it as a listener to a component that generates mouse events, such as a JFrame or a JPanel.
Overall, using adapter classes can simplify the process of handling mouse events in Java, especially when we only need to implement a subset of the available methods.



Examples:
Suppose we have a simple JFrame with a JLabel in the center of the window. We want to display the name of the mouse event whenever the user performs a mouse action, such as clicking or dragging.
To handle these mouse events, we can create a class named MouseEventHandler that extends the MouseAdapter class. Inside this class, we can override the mouseClicked and mouseDragged methods, which will be called whenever the user clicks or drags the mouse, respectively.
In each method, we can get the name of the mouse event using the getName() method of the MouseEvent class, and display it in the center of the JLabel using the drawString() method of the Graphics class.
In the constructor of the MouseEventHandler class, we pass in a JLabel object, which will be used to display the event name.
To use this MouseEventHandler class, we can create an instance of it and add it as a listener to the JFrame or the JLabel.
In this example, we create a JFrame object, add a JLabel object to it, and create an instance of the MouseEventHandler class, passing in the JLabel object.
We then add the MouseEventHandler object as a listener to the JLabel object, using the addMouseListener() and addMouseMotionListener() methods.
Finally, we pack the frame and make it visible.
Now, whenever the user clicks or drags the mouse over the JLabel, the name of the mouse event will be displayed at the center of the JLabel.

import javax.swing.*;
import java.awt.*;
import javax.swing.event.*;
import java.awt.event.*;
class MouseEventPerformer extends JFrame implements MouseListener
{
JLabel l1;
public MouseEventPerformer()
{
setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
setSize(300,300);
setLayout(new FlowLayout(FlowLayout.CENTER));
l1 = new JLabel();
Font f = new Font("Verdana", Font.BOLD, 20);
l1.setFont(f);
l1.setForeground(Color.BLUE);
add(l1);
addMouseListener(this);
setVisible(true);
}
public void mouseExited(MouseEvent m)
{
l1.setText("Mouse Exited");
}
public void mouseEntered(MouseEvent m)
{
l1.setText("Mouse Entered");
}
public void mouseReleased(MouseEvent m)
{
l1.setText("Mouse Released");
}
public void mousePressed(MouseEvent m)
{
l1.setText("Mouse Pressed");
}
public void mouseClicked(MouseEvent m)
{
l1.setText("Mouse Clicked");
}
public static void main(String[] args) {
MouseEventPerformer mep = new MouseEventPerformer();
}
}

import java.awt.*;
import java.awt.event.*;
public class MouseEventDemo extends Frame {
private Label statusLabel;
public MouseEventDemo() {
setLayout(new FlowLayout());
// Create a label to display the event name
statusLabel = new Label("Click or move the mouse");
add(statusLabel);
// Add a mouse listener using adapter classes
addMouseListener(new MouseAdapter() {
public void mouseEntered(MouseEvent e) {
displayEvent("Mouse Entered", e);
}
public void mouseClicked(MouseEvent e) {
displayEvent("Mouse Clicked", e);
}
public void mouseExited(MouseEvent e) {
displayEvent("Mouse Exited", e);
}
});
// Add a mouse motion listener using adapter classes
addMouseMotionListener(new MouseMotionAdapter() {
public void mouseMoved(MouseEvent e) {
displayEvent("Mouse Moved", e);
}
public void mouseDragged(MouseEvent e) {
displayEvent("Mouse Dragged", e);
}
});
// Set up the frame
setTitle("MouseEvent Demo");
setSize(300, 200);
setVisible(true);
}
