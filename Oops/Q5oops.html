OBJECT ORIENTED PROGRAMMING THROUGH JAVA LAB

Q5. Write a Java program for the following:
a) Create a doubly linked list of elements.
b) Delete a given element from the above list.
c) Display the contents of the list after deletion.

Aim: The aim of this experiment is to create a Java program that implements a doubly linked list and performs deletion of a given element. After deletion, the program will display the updated contents of the list.

System and Software Requirements: A Computer with any Operating system with javac compiler installed in it.

Description:
Doubly linked list is a type of data structure that consists of a set of elements or nodes, where each node has a pointer to the previous and next node in the list. In Java, we can create a doubly linked list by defining a Node class that contains the data and pointers to the next and previous nodes.
To create a doubly linked list in Java, we can start by defining a Node class:
class Node {
int data;
Node prev;
Node next;
public Node(int data) {
this.data = data;
this.prev = null;
this.next = null;
}
}
In the above code, we have defined a Node class with three attributes: data, prev, and next. The data attribute represents the value of the node, and the prev and next attributes represent the pointers to the previous and next nodes in the list, respectively.
Next, we can define a LinkedList class that contains the logic to add, delete, and display elements in the doubly linked list:
We have defined a LinkedList class with three methods: add, delete, and display.
The add method adds a new node to the end of the doubly linked list. If the list is empty, the new node becomes the head. Otherwise, we traverse the list to find the last node, and set the next and prev pointers accordingly.
The delete method deletes a node with the given data from the doubly linked list. We traverse the list to find the node with the given data, and then adjust the prev and next pointers of the adjacent nodes to remove the node from the list.
The display method prints the contents of the doubly linked list to the console.
To use the doubly linked list, we can create an instance of the LinkedList class, add some elements to the list, delete a node with a given value, and display the contents of the list:
LinkedList list = new LinkedList();
list.add(1);
list.add(2);
list.add(3);
list.delete(2);
list.display(); // output: 1 3
In the above code, we create a new LinkedList instance, add three elements to the list, delete the node with value 2, and display the remaining elements in the list, which are 1 and 3.

Examples:
For example, let's say we want to create a doubly linked list of integers with the values 1, 2, and 3. We can start by creating the first node, which will be the head of the list:
Node head = new Node(1);
Here, we create a new node with the value of 1 and set it as the head of the list. We can define the Node class as follows:
class Node {
int value;
Node prev;
Node next;
public Node(int value) {
this.value = value;
this.prev = null;
this.next = null;
}
}
This Node class has three attributes: the value of the node and the previous and next nodes. The prev and next nodes are initially set to null because this is the first node in the list.
Next, we can add the second node to the list:
Node second = new Node(2);
head.next = second;
second.prev = head;
Here, we create a new node with the value of 2 and set it as the next node after the head node. We also set the prev pointer of the second node to point to the head node.
Finally, we can add the third node to the list:
Node third = new Node(3);
second.next = third;
third.prev = second;
Here, we create a new node with the value of 3 and set it as the next node after the second node. We also set the prev pointer of the third node to point to the second node.
Now we have a doubly linked list of three nodes with the values 1, 2, and 3, where each node has a reference to the previous and next nodes in the list.

public class DoublyLinkedList {
Node head, tail;
public class Node {
int data;
Node prev, next;
public Node(int data) {
this.data = data;
prev = null;
next = null;
}
}
// Method to add a node to the end of the doubly linked list
public void addNode(int data) {
Node newNode = new Node(data);
if (head == null) {
head = tail = newNode;
head.prev = null;
tail.next = null;
} else {
tail.next = newNode;
newNode.prev = tail;
tail = newNode;
tail.next = null;
}
}
// Method to delete a given element from the doubly linked list
public void deleteNode(int data) {
Node current = head;
while (current != null) {
if (current.data == data) {
if (current == head) {
head = head.next;
head.prev = null;
} else if (current == tail) {
tail = tail.prev;
tail.next = null;
} else {
current.prev.next = current.next;
current.next.prev = current.prev;
}
}
current = current.next;
}
}
// Method to display the contents of the doubly linked list
public void displayList() {
Node current = head;
if (head == null) {
System.out.println("List is empty");
return;
}
System.out.println("Nodes of doubly linked list: ");
while (current != null) {
System.out.print(current.data + " ");
current = current.next;
}
}
public static void main(String[] args) {
DoublyLinkedList dll = new DoublyLinkedList();
dll.addNode(1);
dll.addNode(2);
dll.addNode(3);
dll.addNode(4);
dll.addNode(5);
System.out.println("Original List:");
dll.displayList();
int dataToDelete = 3;
dll.deleteNode(dataToDelete);
System.out.println("\nList after deleting node with data " + dataToDelete + ":");
dll.displayList();
}
}

public class DoublyLinkedList {

public void addNode(int data) {
Node newNode = new Node(data);
if (head == null) {
head = newNode;
} else {
Node current = head;
while (current.next != null) {
current = current.next;
}
current.next = newNode;
newNode.prev = current;
}
}
// Method to delete a given element from the doubly linked list
public void deleteNode(int data) {
if (head == null) {
return;
}
Node current = head;
while (current != null) {
if (current.data == data) {
if (current == head) {
head = current.next;
if (head != null) {
head.prev = null;
}
} else {
current.prev.next = current.next;
if (current.next != null) {
current.next.prev = current.prev;
}
}
return;
}
current = current.next;
}
}

}
