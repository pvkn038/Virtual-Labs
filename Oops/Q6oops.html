Q6. Write a Java program that simulates a traffic light. The program lets the user select one of three lights: red, yellow, or green with radio buttons. On selecting a button, an appropriate message with “Stop” or “Ready” or “Go” should appear above the buttons in selected color. Initially, there is no message shown.

Aim: To create a Java program that simulates a traffic light using radio buttons, where selecting a button will display an appropriate message with the color of the selected button above it.

System and Software Requirements: A Computer with any Operating system with javac compiler installed in it.

Description:
The program will consist of a graphical user interface (GUI) built using Java's Swing library. The GUI will contain three radio buttons for the three traffic light colors: red, yellow, and green. Additionally, there will be a label above the buttons to display the appropriate message based on the selected button.
To implement the program, we will define a class for the GUI window and another class for the traffic light. The traffic light class will contain a state variable to represent the current color of the light. We will use an enumeration to define the three possible values for this variable: RED, YELLOW, and GREEN.
When the user selects a radio button, we will update the traffic light's state variable to match the selected color. Then, we will use a switch statement to determine the appropriate message to display based on the current state of the traffic light. We will set the label's text to this message and also set the label's foreground color to match the selected traffic light color.
We will also include error handling to ensure that the user cannot select multiple radio buttons at once or select an invalid option.
Finally, we will add functionality to reset the traffic light state and label when the user closes the GUI window.
Overall, this program will provide a simple and interactive simulation of a traffic light using Java and Swing.





Examples:
Suppose we have a traffic light simulation program with a GUI containing three radio buttons labeled "Red", "Yellow", and "Green", and a label above the buttons. Initially, the label shows no message.
When the user selects the "Red" radio button, the program updates the traffic light state variable to RED and sets the label text to "Stop" in red color.
When the user selects the "Yellow" radio button, the program updates the traffic light state variable to YELLOW and sets the label text to "Ready" in yellow color.
When the user selects the "Green" radio button, the program updates the traffic light state variable to GREEN and sets the label text to "Go" in green color.
If the user tries to select multiple radio buttons at once, the program will display an error message and prevent the selection.
When the user closes the GUI window, the program will reset the traffic light state variable to null and clear the label text.

import javax.swing.*;
import java.awt.*;
import java.awt.event.*;
public class TrafficLight implements ActionListener {
JFrame frame;
JRadioButton redButton, yellowButton, greenButton;
JLabel label;
public TrafficLight() {
// Create the frame
frame = new JFrame("Traffic Light");
frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
frame.setSize(250, 150);
// Create the radio buttons
redButton = new JRadioButton("Red");
yellowButton = new JRadioButton("Yellow");
greenButton = new JRadioButton("Green");
// Create a button group for the radio buttons
ButtonGroup buttonGroup = new ButtonGroup();
buttonGroup.add(redButton);
buttonGroup.add(yellowButton);
buttonGroup.add(greenButton);
// Create the label
label = new JLabel("");
label.setFont(new Font("Arial", Font.BOLD, 24));
label.setHorizontalAlignment(JLabel.CENTER);
// Add the radio buttons and label to the frame
JPanel panel = new JPanel();
panel.setLayout(new GridLayout(4, 1));
panel.add(redButton);
panel.add(yellowButton);
panel.add(greenButton);
panel.add(label);
frame.add(panel);
// Add action listeners to the radio buttons
redButton.addActionListener(this);
yellowButton.addActionListener(this);
greenButton.addActionListener(this);
// Show the frame
frame.setVisible(true);
}
public void actionPerformed(ActionEvent e) {
if (e.getSource() == redButton) {
label.setText("Stop");
label.setForeground(Color.RED);
} else if (e.getSource() == yellowButton) {
label.setText("Ready");
label.setForeground(Color.YELLOW);
} else if (e.getSource() == greenButton) {
label.setText("Go");
label.setForeground(Color.GREEN);
}
}
public static void main(String[] args) {
new TrafficLight();
}
}

import javax.swing.*;

// Create the label to display the message
JLabel label = new JLabel("");

label.setHorizontalAlignment(JLabel.CENTER);
frame.add(label, BorderLayout.NORTH);
// Create the panel for the radio buttons
JPanel radioPanel = new JPanel(new GridLayout(3, 1));
frame.add(radioPanel, BorderLayout.CENTER);
// Create the radio buttons
JRadioButton redButton = new JRadioButton("Red");
JRadioButton yellowButton = new JRadioButton("Yellow");
JRadioButton greenButton = new JRadioButton("Green");
// Create a button group for the radio buttons
ButtonGroup buttonGroup = new ButtonGroup();
buttonGroup.add(redButton);
buttonGroup.add(yellowButton);
buttonGroup.add(greenButton);
// Add the radio buttons to the panel
radioPanel.add(redButton);
radioPanel.add(yellowButton);
radioPanel.add(greenButton);
// Add an action listener to the radio buttons
ActionListener actionListener = new ActionListener() {
public void actionPerformed(ActionEvent e) {
if (e.getSource() == redButton) {
label.setText("Stop");
label.setForeground(Color.RED);
} else if (e.getSource() == yellowButton) {
label.setText("Ready");
label.setForeground(Color.YELLOW);
} else if (e.getSource() == greenButton) {
label.setText("Go");
label.setForeground(Color.GREEN);
}
}
};
redButton.addActionListener(actionListener);
yellowButton.addActionListener(actionListener);
greenButton.addActionListener(actionListener);
// Show the frame
frame.setVisible(true);
}
}
