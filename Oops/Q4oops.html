Q4. Write a Java program that implements a multi-thread application that has three threads. First thread generates random integer every 1 second and if the value is even, second thread computes the square of the number and prints. If the value is odd, the third thread will print the value of cube of the number.

Aim: To develop a Java program that implements a multi-threaded application consisting of three threads, where the first thread generates a random integer every second, and if the value is even, it communicates with the second thread to compute and print the square of the number, and if the value is odd, it communicates with the third thread to print the cube of the number. The program should demonstrate the use of multi-threading concepts in Java and handle thread communication efficiently.

System and Software Requirements: A Computer with any Operating system with javac compiler installed in it.

Description:
The Java program we are testing is a multi-threaded application that involves three threads. The first thread is responsible for generating random integers every second. The second thread checks if the number is even, calculates the square of the number and prints the result. The third thread checks if the number is odd, calculates the cube of the number and prints the result.
The experiment aims to verify the correctness of the program and to measure its performance. We will create a test plan that involves running the program with different input values and measuring the output produced by each thread.
Test Plan:
1. Input Test: We will test the program with different input values to ensure that the program produces the correct output for each scenario. We will test with a range of integer values from -100 to 100 to test the program's ability to handle positive and negative values.
2. Performance Test: We will measure the program's performance by measuring the time it takes to generate random numbers and perform the computations. We will use a timer to measure the elapsed time between the start and end of the program.
3. Stress Test: We will test the program's ability to handle a large number of requests by generating a large number of random numbers and measuring the program's ability to handle the load.
4. Error Handling Test: We will test the program's ability to handle errors by passing invalid inputs to the program, such as non-integer values, and measuring how the program responds.
5. Concurrency Test: We will test the program's ability to handle concurrency by simulating multiple threads generating and processing numbers simultaneously.
Expected Results:
1. Input Test: The program should produce the correct output for each input value. For even numbers, the program should calculate the square of the number and print the result. For odd numbers, the program should calculate the cube of the number and print the result.
2. Performance Test: The program should generate and process numbers within a reasonable time frame. The elapsed time should be within an acceptable range.
3. Stress Test: The program should be able to handle a large number of requests without crashing or producing errors.
4. Error Handling Test: The program should handle errors gracefully by producing an error message or terminating gracefully.
5. Concurrency Test: The program should be able to handle multiple threads generating and processing numbers simultaneously without producing errors or unexpected results.



Examples:
Suppose the first thread generates the following sequence of random integers:
1, 4, 7, 10, 12, 16
The second thread, which checks for even numbers, will calculate the square of 4, 10, and 12 and print the results:
4 squared is 16
10 squared is 100
12 squared is 144
The third thread, which checks for odd numbers, will calculate the cube of 1, 7, and 16 and print the results:
1 cubed is 1
7 cubed is 343
16 cubed is 4096
So the final output of the program would be:
4 squared is 16
1 cubed is 1
10 squared is 100
7 cubed is 343
12 squared is 144
16 cubed is 4096
This is just a simple example, but it illustrates how the program would work in practice.

import java.util.Random;
class RandomNumberThread extends Thread {
public void run() {
Random random = new Random();
for (int i = 0; i < 10; i++) {
int randomInteger = random.nextInt(100);
System.out.println("Random Integer generated : " + randomInteger);
if((randomInteger%2) == 0) {
SquareThread sThread = new SquareThread(randomInteger);
sThread.start();
}
else {
CubeThread cThread = new CubeThread(randomInteger);
cThread.start();
}
try {
Thread.sleep(1000);
}
catch (InterruptedException ex) {
System.out.println(ex);
}
}
}
}
class SquareThread extends Thread {
int number;
SquareThread(int randomNumbern) {
number = randomNumbern;
}
public void run() {
System.out.println("Square of " + number + " = " + (number * number));
}
}
class CubeThread extends Thread {
int number;
CubeThread(int randomNumber) {
number = randomNumber;
}
public void run() {
System.out.println("Cube of " + number + " = " + number * number * number);
}
}

import java.util.LinkedList;
import java.util.Queue;
import java.util.Random;
public class Main {
public static void main(String[] args) {
Queue queue = new LinkedList<>();
Random random = new Random();
Thread t1 = new Thread(() -> {
while (true) {
int num = random.nextInt(100);
System.out.println("Generated number: " + num);
synchronized (queue) {
queue.add(num);
queue.notifyAll();
}
try {
Thread.sleep(1000);
} catch (InterruptedException e) {
e.printStackTrace();
}
}
});
Thread t2 = new Thread(() -> {
while (true) {
synchronized (queue) {
while (queue.isEmpty() || queue.peek() % 2 != 0) {
try {
queue.wait();
} catch (InterruptedException e) {
e.printStackTrace();
}
}
int num = queue.remove();
int square = num * num;
System.out.println("Square of " + num + " is " + square);
}
}
});
Thread t3 = new Thread(() -> {
while (true) {
synchronized (queue) {
while (queue.isEmpty() || queue.peek() % 2 == 0) {
try {
queue.wait();
} catch (InterruptedException e) {
e.printStackTrace();
}
}
int num = queue.remove();
int cube = num * num * num;
System.out.println("Cube of " + num + " is " + cube);
}
}
});
t1.start();
t2.start();
t3.start();
} }
