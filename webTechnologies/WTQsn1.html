Q1. Write an HTML page including JavaScript that takes a given set of integer numbers andshows them after sorting in descendingorder..

Aim: To write an HTML page including JavaScript that takes a given set of integer numbers andshows them after sorting in descendingorder

Description:

This HTML page contains an input field where you can enter a set of numbers separated by commas (e.g., 5, 3, 8, 1). When you click the "Sort" button, the JavaScript function sortNumbersDescending() is triggered. It takes the input value, splits it into an array of numbers, converts them to integers, sorts them in descending order using the sort() method, and then joins them back into a string. The sorted numbers are displayed in the result paragraph element.

This is a basic HTML structure for the page. The head section contains the title of the page, and we have an empty script tag, where we'll write our JavaScript code. The actual HTML content goes inside the body section.

In the body section, we have a heading h1 that displays the title of our page, which is "Sort Numbers Descending". We also have a paragraph that provides instructions to the user, asking them to enter a set of numbers separated by commas.

Next, we have an input element with the type set to "text" and an id attribute set to "numbers". This input field allows the user to enter the numbers they want to sort. We've also added a placeholder text to provide an example format.

After that, we have a button element with the onclick attribute set to "sortNumbersDescending()". This button triggers the JavaScript function sortNumbersDescending() when clicked.

Lastly, we have a p element with the id set to "result". This element will be used to display the sorted numbers to the user.

Disclaimer : We recommend you to try the code by yourself before checking the solution.

<!DOCTYPE html>
<html>
<head>
<title>Sort Numbers Descending
<script>
function sortNumbersDescending() {
// Get the input value var numbersInput = document.getElementById("numbers").value;
// Split the input by comma to get individual numbers var numbersArray = numbersInput.split(",");
// Convert the strings to numbers for (var i = 0; i < numbersArray.length; i++) {
numbersArray[i] = parseInt(numbersArray[i]);
}
// Sort the numbers in descending order numbersArray.sort(function(a, b) {
return b - a;
});
// Join the sorted numbers into a string var sortedNumbers = numbersArray.join(", ");
// Display the sorted numbers document.getElementById("result").innerHTML = "Sorted Numbers: " + sortedNumbers;
}
</script> </head> <body>
