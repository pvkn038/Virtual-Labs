Q2. Write an HTML page including any required Javascript that takes a number from one text field in the range of 0 to 999 and shows it in another text field in words. If the number is out of range, it should show ―out of range‖ and if it is not a number, it should show ―not a number‖ message in the result box.

Aim: To write an HTML page including any required Javascript that takes a number from one text field in the range of 0 to 999 and shows it in another text field in words. If the number is out of range, it should show ―out of range‖ and if it is not a number, it should show ―not a number‖ message in the result box.

Description:

The HTML page consists of the following elements:

  • Heading: "Number to Words Converter"
  • Paragraph: "Enter a number between 0 and 999:"
  • Input field: Allows the user to enter a number
  • Button: "Convert" - triggers the conversion process
  • Paragraph: "Result:" - indicating the result section
  • Result input field: Displays the converted number in words
  • JavaScript function: convertToWords() - performs the conversion logic
When the user enters a number and clicks the "Convert" button, the JavaScript function convertToWords() is called. It retrieves the entered number, checks if it is a valid number, within the range of 0 to 999. If the input is not a number, it displays "not a number" in the result field. If the number is out of range, it displays "out of range". Otherwise, it converts the number to its word representation and displays it in the result field.

Disclaimer : We recommend you to try the code by yourself before checking the solution.

Try on CodeEditor

<!DOCTYPE html>
<html>
<head>
<title>Number to Words Converter</title>
<script>
function convertToWords() {
// Get the input value var inputNumber = document.getElementById("numberInput").value;
// Convert the input to a number var number = Number(inputNumber);
// Check if the number is within the range of 0 to 999 if (number >= 0 && number <= 999) {
var words = convertNumberToWords(number);
document.getElementById("result").value = words;
} else if (isNaN(number)) {
document.getElementById("result").value = "Not a number";
} else {
document.getElementById("result").value = "Out of range";
}
}
function convertNumberToWords(number) {
var units = ['', 'one', 'two', 'three', 'four', 'five', 'six', 'seven', 'eight', 'nine'];
var tens = ['', '', 'twenty', 'thirty', 'forty', 'fifty', 'sixty', 'seventy', 'eighty', 'ninety'];
var teens = ['ten', 'eleven', 'twelve', 'thirteen', 'fourteen', 'fifteen', 'sixteen', 'seventeen', 'eighteen', 'nineteen'];
var words = '';
if (number === 0) {
words = 'zero';
} else if (number < 10) {
words = units[number];
} else if (number < 20) {
words = teens[number - 10];
} else if (number < 100) {
words = tens[Math.floor(number / 10)] + ' ' + units[number % 10];
} else {
words = units[Math.floor(number / 100)] + ' hundred ' + convertNumberToWords(number % 100);
}
return words.trim();
}
</script>
</head>
<body>
<h1>Number to Words Converter</h1>
<p>Enter a number between 0 and 999:</p>
<input type="text" id="numberInput" placeholder="e.g., 123">
<button onclick="convertToWords()">Convert</button>
<p>Result:</p>
<input type="text" id="result" readonly>
</body>
</html>
