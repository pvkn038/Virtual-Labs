Q12. Write a Python program to check if a number is positive, negative or zero.

Aim: A Python program to check if a number is positive, negative or zero.

System and Software Requirements: A Computer with an python interpreter.

Description:
Positive Numbers: Any number above zero is known as a positive number.
Positive numbers are written without any sign or a '+' sign in front of them and they are counted up from zero, i.e. 1, + 2, 3, +4 etc.
Negative Numbers: Any number below zero is known as a negative number.
Negative numbers are always written with a 'âˆ’' sign in front of them and they are counted down from zero.
i.e. -1, -2, -3, -4 etc.
Always look at the sign in front of a number to check if it is positive or negative.
Zero, 0, is neither positive nor negative.
Algorithm:
Step-1:Prompt user to enter a number and store it in num.
Step-2:If num is greater than 0, print "Positive number".
Step-3:If num is 0, print "Zero".
Step-4:If num is less than 0, print "Negative number".
Step-5:End the program.
