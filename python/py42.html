Q42. Find the determinant for a 2 * 2 matrix using scipy library module.

Aim: To find the determinant for a 2 * 2 matrix using scipy library module.

System and Software Requirements: A Computer with an python interpreter.

Description:
The determinant value can be calculated on a matrix or on an array that has more than one dimension. It may sometimes be required to understand the marix/array better. This is where the determinant operation would be needed.
SciPy offers a function named ‘det’ that is present in the ‘linalg’ class which is short for ‘Linear Algebra’.
Algorithm:
Step-1: Import scipy and numpy packages.
Step-2: Create a numpy array 'two_d_matrix' with given matrix values.
Step-3: Use 'linalg.det()' function to calculate the determinant of the matrix by passing the 'two_d_matrix' array as argument and store the result in a variable.
Step-4: Print the determinant value to display the result.

Disclaimer : We recommend you to try the code by yourself before checking the solution.

Try on CodeEditor

from scipy import linalg
import numpy as np
two_d_matrix = np.array([ [7, 9], [33, 8] ])
print("The determinant value of the matrix is :")
print(linalg.det(two_d_matrix ))

import scipy
import numpy as np
X = np.array([[5,1],[8,4]])
M = scipy.linalg.det(X)
#Printing the result
print('Determinant of {} is {}'.format(X,M))
