Q14. Write a Python program to compute the greatest common divisor (GCD) of two positive integers.

Aim: A Python program to compute the greatest common divisor (GCD) of two positive integers.

System and Software Requirements: A Computer with an python interpreter.

Description:
Approach 1:
Algorithm:
Step-1: Initialize GCD = 1.
Step-2: Run a loop in the iteration of (i) between [1, min(num1, num2)].
Step-3: Note down the highest number that divides both num1 & num2.
Step-4: If i satisfies (num1 % i == 0 and num2 % i == 0) then new value of GCD is i.
Step-5: Print value of GCD.

Approach 2:
Algorithm:
Step-1: Run a while loop until num1 is not equals to num2.
Step-2: If num1>num2 then num1 = num1 – num2.
Step-3: Else num2 = num2 – num1.
Step-4: After the loop ends both num1 & num2 stores GCD.


Disclaimer : We recommend you to try the code by yourself before checking the solution.

Try on CodeEditor

num1 = 36
num2 = 60
gcd = 1
for i in range(1, min(num1, num2)):
  if num1 % i == 0 and num2 % i == 0:
   gcd = i
print("GCD of", num1, "and", num2, "is", gcd)
