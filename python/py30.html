Q30. Write a Python program to sort a list of elements using the bubble sort algorithm

Aim: A Python program to sort a list of elements using the bubble sort algorithm.

System and Software Requirements: A Computer with an python interpreter.

Description:
The bubble sort uses a straightforward logic that works by repeating swapping the adjacent elements if they are not in the right order. It compares one pair at a time and swaps if the first element is greater than the second element; otherwise, move further to the next pair of elements for comparison.
Algorithm:
Step-1: #Creating a bubble sort function.
Step-2: def bubble_sort(list1):
Step-3: #Outer loop for traverse the entire list.
Step-4: for i in range(0,len(list1)-1):
Step-5: for j in range(len(list1)-1):
Step-6: if(list1[j]>list1[j+1]):
Step-7: temp = list1[j].
Step-8: list1[j] = list1[j+1].

def bubble_sort(list1):
# Outer loop for traverse the entire list
 for i in range(0,len(list1)-1):
  for j in range(len(list1)-1):
   if(list1[j]>list1[j+1]):
    temp = list1[j]
    list1[j] = list1[j+1]
    list1[j+1] = temp
 return list1
list1 = [5, 3, 8, 6, 7, 2]
print("The unsorted list is: ", list1)
# Calling the bubble sort function
print("The sorted list is: ", bubble_sort(list1))
