Q39. Write a Python class which has two methods get_String and print_String.get_Stringaccepta string from the user and print_String print the string in uppercase.

Aim: A Python class which has two methods get_String and print_String.get_Stringaccepta string from the user and print_String print the string in uppercase.

System and Software Requirements: A Computer with an python interpreter.

Description:
Algorithm:
Step-1: Define a class "IOString" with two methods - "get_String()" and "print_String()".
Step-2: Initialize a class variable "str1" as an empty string in the constructor using "init()".
Step-3: Get the string input from the user using "input()" and store it in the "str1" variable using the "get_String()" method.
Step-4: Print the uppercase version of the string using the "print_String()" method and the "upper()" function.
Step-5: Instantiate an object of the "IOString" class, call the "get_String()" and "print_String()" methods on the object to get the user input and print its uppercase version.

Disclaimer : We recommend you to try the code by yourself before checking the solution.

Try on CodeEditor

class IOString():
  def __init__(self):
   self.str1 = ""
  def get_String(self):
   self.str1 = input()
  def print_String(self):
   print(self.str1.upper())
str1 = IOString()
str1.get_String()
str1.print_String()
