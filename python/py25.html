Q25. Write a Python program to count and display the vowels of a given text.

Aim: A Python program to count and display the vowels of a given text.

System and Software Requirements: A Computer with an python interpreter.

Description:
Approach:
Algorithm:
Step-1: Take a string from the user and store it in a variable.
Step-2: Initialize a count variable to 0.
Step-3: Use a for loop to traverse through the characters in the string.
Step-4: Use an if statement to check if the character is a vowel or not and increment the count variable if it is a vowel.
Step-5: Print the total number of vowels in the string.
Step-6: Exit.


Disclaimer : We recommend you to try the code by yourself before checking the solution.

# Python code to count and display number of vowels
# Simply using for and comparing it with a
# string containing all vowels
def Check_Vow(string, vowels):
   final = [each for each in string if each in vowels]
   print(len(final))
# Driver Code
string = "Geeks for Geeks"
vowels = "AaEeIiOoUu"
Check_Vow(string, vowels);

string=raw_input("Enter string:")
vowels=0
for i in string:
   if(i=='a' or i=='e' or i=='i' or i=='o' or i=='u' or i=='A' or i=='E' or i=='I' or i=='O' or i=='U'):
    vowels=vowels+1
print("Number of vowels are:")
print(vowels)
