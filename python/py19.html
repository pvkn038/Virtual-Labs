Q19. Write a Python function that checks whether a passed string is palindrome or not

Aim: A Python function that checks whether a passed string is palindrome or not

System and Software Requirements: A Computer with an python interpreter

Description:
Algorithm:
Step-1: Input the string.
Step-2: Find the reverse of the string.
Step-3: If the reverse of the string is equal to the input string, then return true. Else, return false.


Disclaimer : We recommend you to try the code by yourself before checking the solution.

Try on CodeEditor

def isPalindrome(string):
  left_pos = 0
  right_pos = len(string) - 1
  while right_pos >= left_pos:
   if not string[left_pos] == string[right_pos]:
    return False
   left_pos += 1
   right_pos -= 1
  return True
print(isPalindrome('aza'))

# function which return reverse of a string
def isPalindrome(s):
  return s == s[::-1]
# Driver code
s = "malayalam"
ans = isPalindrome(s)
if ans:
  print("Yes")
else:
  print("No")
