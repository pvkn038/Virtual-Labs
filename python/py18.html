Q18. Write a Python function that accepts a string and calculate the number of upper case letters and lower case letters.

Aim: A Python function that accepts a string and calculate the number of upper case letters and lower case letters.

System and Software Requirements: A Computer with an python interpreter.

Description:
Uppercase: Capital letters.
Lowercase: Small letters.
Algorithm:
Step-1: Define a function called string_test that takes a single argument s.
Step-2: Initialize a dictionary d with keys "UPPER_CASE" and "LOWER_CASE" and values set to 0.
Step-3: Loop through each character in the string s using a for loop.
Step-4: If the character is uppercase, increment the value associated with the "UPPER_CASE" key in the dictionary d; if the character is lowercase, increment the value associated with the "LOWER_CASE" key in d.
Step-5: Print the original string s, and the counts of upper and lowercase characters as computed and stored in the dictionary d.

def string_test(s):
  d={"UPPER_CASE":0, "LOWER_CASE":0}
  for c in s:
   if c.isupper():
    d["UPPER_CASE"]+=1
   elif c.islower():
    d["LOWER_CASE"]+=1
   else:
    pass
print ("Original String : ", s)
print ("No. of Upper case characters : ", d["UPPER_CASE"])
print ("No. of Lower case Characters : ", d["LOWER_CASE"])
string_test('The quick Brown Fox')

def upperlower(string):
  upper = 0
  lower = 0
  for i in range(len(string)):
# For lower letters
   if (ord(string[i]) >= 97 and
    ord(string[i]) <= 122):
    lower += 1
# For upper letters
   elif (ord(string[i]) >= 65 and
    ord(string[i]) <= 90):
    upper += 1
  print('Lower case characters = %s' %lower,
   'Upper case characters = %s' %upper)
