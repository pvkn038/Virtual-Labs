Q22. Write a Python program to get the factorial of a non-negative integer using recursion

Aim: A Python program to get the factorial of a non-negative integer using recursion

System and Software Requirements: A Computer with an python interpreter.

Description:
Factorial of a non-negative integer, is multiplication of all integers smaller than or equal to n.
For example factorial of 6 is 6*5*4*3*2*1 which is 720.
Algorithm:
Step-1: Define a function called factorial(n) that returns the factorial of n using recursion.
Step-2: If n is 0, return 1 (the base case).
Step-3: Otherwise, recursively call the factorial function with argument n-1 and return the product of n and the result.
Step-4: Prompt the user to input a number to compute its factorial, and assign the value to n.
Step-5: Print the result of calling the factorial function with n as the argument.

def factorial(n):
  if n < 0:
   return 0
  elif n == 0 or n == 1:
   return 1
  else:
   fact = 1
  while(n > 1):
   fact *= n
   n -= 1
  return fact
