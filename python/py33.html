Q33. Write a Python program to print last n lines of a file.

Aim:A Python program to print last n lines of a file.

System and Software Requirements: A Computer with an python interpreter.

Description:
Text file name: Sample.txt
First line
Second line
Third line
Fourth line
Fifth line
Sixth line
Seventh line
Eighth line
Nineth line
Tenth line.

Algorithm:
Step-1: Define a function "LastNlines" that takes two parameters, the name of the file to read and the number of lines to print.
Step-2: Open the file and read its lines using "readlines()" method.
Step-3: Print the last "N" lines of the file using the list slice notation "[-N:]".
Step-4: In the main block, specify the file name and number of lines to print.
Step-5: Call the "LastNlines" function with the file name and number of lines as arguments and catch any exceptions that might occur.


Disclaimer : We recommend you to try the code by yourself before checking the solution.

def LastNlines(fname, N):
 with open(fname) as file:
  for line in (file.readlines() [-N:]):
   print(line, end ='')

import os
def LastNlines(fname, N):
 bufsize = 8192
 fsize = os.stat(fname).st_size
 iter = 0
 with open(fname) as f:
  if bufsize > fsize:
  bufsize = fsize-1
  fetched_lines = []
  while True:
   iter += 1
   f.seek(fsize-bufsize * iter)
   fetched_lines.extend(f.readlines())
   if len(fetched_lines) >= N or f.tell() == 0:
    print(''.join(fetched_lines[-N:]))
    break
if __name__ == '__main__':
 fname = 'File1.txt'
 N = 3
 try:
  LastNlines(fname, N)
 except:
  print('File not found')
