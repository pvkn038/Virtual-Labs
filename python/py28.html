Q28. Write a program combine lists that combines these lists into a dictionary.

Aim: A program combine lists that combines these lists into a dictionary.

System and Software Requirements: A Computer with an python interpreter

Description:
Approach:
The elements of the first one serve as keys and the elements of the second one serve as values. Each item in the first list must be unique and hashable. Use zip() in combination with dict() to combine the values of the two lists into a dictionary.
Algorithm:
Step-1: Define a function called test that takes two arguments, keys and values.
Step-2: Inside the function, create a dictionary by calling the zip() function with keys and values as arguments.
Step-3: Return the resulting dictionary.
Step-4: Define two lists called l1 and l2 with values ['a', 'b', 'c', 'd', 'e', 'f'] and [1, 2, 3, 4, 5], respectively.
Step-5: Print the original lists, and then print the dictionary resulting from calling the test function with l1 and l2 as arguments.

Disclaimer : We recommend you to try the code by yourself before checking the solution.

Try on CodeEditor

def test(keys, values):
  return dict(zip(keys, values))
l1 = ['a', 'b', 'c', 'd', 'e', 'f']
l2 = [1, 2, 3, 4, 5]
print("Original lists:")
print(l1)
print(l2)
print("\nCombine the values of the said two lists into a dictionary:")
print(test(l1, l2))
