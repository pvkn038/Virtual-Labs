Q40. Write a Python class named Rectangle constructed by length and width and a method which will compute the area of a rectangle.

Aim: A Python class named Rectangle constructed by length and width and a method which will compute the area of a rectangle.

System and Software Requirements: A Computer with an python interpreter.

Description:
In Euclidean plane geometry, a rectangle is a quadrilateral with four right angles. To find the area of a rectangle,multiply the length by the width. A rectangle with four sides of equal length is a square. Following image represents the area of a rectangle.
Algorithm:
Step-1: Take the value of length and breadth from the user.
Step-2: Create a class and using a constructor initialise values of that class.
Step-3: Create a method called as area and return the area of the class.
Step-4: Create an object for the class.
Step-5: Using the object, call the method area() with the parameters as the length and breadth taken from the user.
Step-6: Print the area.
Step-7: Exit.

class Rectangle():
  def __init__(self, l, w):
   self.length = l
   self.width = w
  def rectangle_area(self):
   return self.length*self.width
newRectangle = Rectangle(12, 10)
print(newRectangle.rectangle_area())

class rectangle():
  def __init__(self,breadth,length):
   self.breadth=breadth
   self.length=length
  def area(self):
   return self.breadth*self.length
a=int(input("Enter length of rectangle: "))
b=int(input("Enter breadth of rectangle: "))
