Q15. Write a Python program Make a Simple Calculator

Aim: A Python program to Make a Simple Calculator.

System and Software Requirements: A Computer with an python interpreter

Description:
Approach:
we can create a simple calculator for performing the different arithmetical operations,
such as addition, subtraction, multiplication, and division.
Algorithm:
Step-1: We can choose the desired operation from the option of a, b, c, and d.
Step-2: We can take two numbers, and if… elif… else, branching is used for executing the particular operation.
Step-3: We will use add(), subtract(), multiply() and divide() function for evaluation the respective operation in the calculator.
