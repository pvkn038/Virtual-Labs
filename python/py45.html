Q45. Using matplot lib and scipy libraries, apply the following operations on an image.
a) Display the image crop image
b) flip
c) rotate
d) Display the statistical information of the image
e) Turn upside down.

Aim: The aim of this experiment is to use Matplotlib and Scipy libraries to apply several operations on an image, including cropping, flipping, rotation, statistical information display, and turning the image upside down. These operations will help to demonstrate how these libraries can be used to manipulate and analyze digital images.

System and Software Requirements: A Computer with an python interpreter.

Description:
The objective of this experiment is to demonstrate the use of Matplotlib and Scipy libraries in performing various image processing operations on an image. In this experiment, we will perform the following operations on an image:
a) Display the image crop image
b) Flip
c) Rotate
d) Display the statistical information of the image
e) Turn upside down
For this experiment, we will use the Python programming language along with the Matplotlib and Scipy libraries.
The experiment will involve the following steps:
Load the image using the imread function from the scipy library.
Display the original image using the imshow function from the matplotlib library. Crop the image using the slicing operator in NumPy and display the cropped image using imshow.
Flip the image using the flip function from the numpy library and display the flipped image using imshow.
Rotate the image using the rotate function from the scipy library and display the rotated image using imshow.
Display the statistical information of the image using the mean, median, standard deviation, and variance functions from the numpy library.
Turn the image upside down without program.
Save the final image using the imsave function from the scipy library.
The above steps will be implemented in a Python program using the Matplotlib and Scipy libraries. The program will take an input image, perform the above operations, and display the results. This experiment will help in understanding the different image processing techniques using Python and how to use Matplotlib and Scipy libraries for image processing.

import matplotlib.pyplot as plt
from scipy import ndimage
from scipy import misc
# Load the image
image = misc.ascent()
# Display the image
plt.imshow(image, cmap=plt.cm.gray)
plt.show()
# Crop the image
cropped_image = image[100:300, 100:300]
# Display the cropped image
plt.imshow(cropped_image, cmap=plt.cm.gray)
plt.show()
# Flip the image
flipped_image = ndimage.rotate(image, 180)
# Display the flipped image
plt.imshow(flipped_image, cmap=plt.cm.gray)
plt.show()
# Rotate the image
rotated_image = ndimage.rotate(image, 45)
# Display the rotated image
plt.imshow(rotated_image, cmap=plt.cm.gray)
plt.show()
# Display the statistical information of the image
print("Image minimum value: ", image.min())
print("Image maximum value: ", image.max())
print("Image mean value: ", image.mean())
# Turn the image upside down
upside_down_image = ndimage.rotate(image, 180)
# Display the upside down image
plt.imshow(upside_down_image, cmap=plt.cm.gray)
plt.show()

import cv2
import numpy as np
from matplotlib import pyplot as plt
# Load the image using OpenCV
image = cv2.imread('example_image.jpg', cv2.IMREAD_GRAYSCALE)
# Display the image
plt.imshow(image, cmap='gray')
plt.show()
# Crop the image
cropped_image = image[100:300, 100:300]
# Display the cropped image
plt.imshow(cropped_image, cmap='gray')
plt.show()
# Flip the image
flipped_image = cv2.flip(image, 0)
# Display the flipped image
plt.imshow(flipped_image, cmap='gray')
plt.show()
# Rotate the image
(rows, cols) = image.shape
M = cv2.getRotationMatrix2D((cols/2,rows/2), 45, 1)
rotated_image = cv2.warpAffine(image, M, (cols,rows))
# Display the rotated image
plt.imshow(rotated_image, cmap='gray')
plt.show()
# Display the statistical information of the image
print("Image minimum value: ", image.min())
print("Image maximum value: ", image.max())
print("Image mean value: ", image.mean())
# Turn the image upside down
upside_down_image = cv2.flip(image, 1)
# Display the upside down image
plt.imshow(upside_down_image, cmap='gray')
plt.show()
