Q44. Draw a barchart with the following data using matplot lib Men_mean=[20,35,30,35,27] Women_mean=[25,32,34,20,25], Men_std=[2,3,4,1,2] Women_std=[3,5,2,3,3].

Aim: To draw a barchart with the following data using matplot lib Men_mean=[20,35,30,35,27] Women_mean=[25,32,34,20,25], Men_std=[2,3,4,1,2] Women_std=[3,5,2,3,3].

System and Software Requirements: A Computer with an python interpreter.

Description:
To draw a bar chart for the given data using Matplotlib without programming, you can follow the following steps:
Start by defining the x-axis values as the categories that we have. In this case, the categories are not given, so we can use numbers as the categories. Let's define the categories as 0, 1, 2, 3, and 4.
Next, we need to define the y-axis values for men and women. We have the mean values and standard deviation values for men and women. We can use the mean values as the y-axis values and the standard deviation values to add error bars to the bar chart.
Now, we can create a bar chart with two bars for each category, one for men and one for women. The height of each bar will be the mean value, and the error bars will show the standard deviation.
Finally, we can add labels for the x-axis, y-axis, and title to the chart to make it more informative.
So, based on the above steps, we can draw a bar chart with the given data as follows:
X-axis Categories: 0, 1, 2, 3, 4
Y-axis Values for Men: [20, 35, 30, 35, 27]
Y-axis Values for Women: [25, 32, 34, 20, 25]
Error Bars for Men: [2, 3, 4, 1, 2]
Error Bars for Women: [3, 5, 2, 3, 3].

import matplotlib
import matplotlib.pyplot as plt
import numpy as np
men_means, men_std = (20, 35, 30, 35, 27), (2, 3, 4, 1, 2)
women_means, women_std = (25, 32, 34, 20, 25), (3, 5, 2, 3, 3)
ind = np.arange(len(men_means)) # the x locations for the groups
width = 0.35 # the width of the bars
fig, ax = plt.subplots()
rects1 = ax.bar(ind - width/2, men_means, width, yerr=men_std, label='Men')
rects2 = ax.bar(ind + width/2, women_means, width, yerr=women_std, label='Women')
# Add some text for labels, title and custom x-axis tick labels, etc.
ax.set_ylabel('Scores')
ax.set_title('Scores by group and gender')
ax.set_xticks(ind)
ax.set_xticklabels(('G1', 'G2', 'G3', 'G4', 'G5'))
ax.legend()
def autolabel(rects, xpos='center'):
""" Attach a text label above each bar in *rects*, displaying its height. *xpos* indicates which side to place the text w.r.t. the center of the bar. It can be one of the following {'center', 'right', 'left'}. """
ha = {'center': 'center', 'right': 'left', 'left': 'right'}
offset = {'center': 0, 'right': 1, 'left': -1}
for rect in rects:
height = rect.get_height()
ax.annotate('{}'.format(height),
xy=(rect.get_x() + rect.get_width() / 2, height),
xytext=(offset[xpos]*3, 3), # use 3 points offset
textcoords="offset points", # in both directions
ha=ha[xpos], va='bottom')
autolabel(rects1, "left")
autolabel(rects2, "right")
fig.tight_layout()
plt.show()

import numpy as np
import matplotlib.pyplot as plt
# Data
Men_mean = [20,35,30,35,27]
Women_mean = [25,32,34,20,25]
Men_std = [2,3,4,1,2]
Women_std = [3,5,2,3,3]
x_labels = ['A', 'B', 'C', 'D', 'E']
# Set the bar width
bar_width = 0.35
# Calculate the positions of the bars on the x-axis
r1 = np.arange(len(Men_mean))
r2 = [x + bar_width for x in r1]
# Plot the bars for Men and Women
plt.bar(r1, Men_mean, width=bar_width, yerr=Men_std, color='blue', edgecolor='black', capsize=7, label='Men')
plt.bar(r2, Women_mean, width=bar_width, yerr=Women_std, color='pink', edgecolor='black', capsize=7, label='Women')
# Add xticks on the middle of the group bars
plt.xlabel('Group', fontweight='bold')
plt.xticks([r + bar_width/2 for r in range(len(Men_mean))], x_labels)
# Add ylabel and title
plt.ylabel('Scores')
plt.title('Scores by group and gender')
# Add legend
plt.legend()
# Show the chart
plt.show()
