Q8. Write a Python program to count the number 6 in a given list.

Aim: A Python program to count the number 6 in a given list.

System and Software Requirements: A Computer with an interpreter.

Description:
Approach 1:
The said code defines a function called "list_count_6" that takes a list of numbers as its argument, "nums". Inside the function, it initializes a variable "count" to zero. Next, it uses a for loop to iterate through each element in the "nums" list. For each iteration, it checks if the current element i.e. num is equal to 6, if true it increments the "count" by 1. After the for loop completes, the function returns the final "count" of how many times the number 6 appears in the list.
Approach 2:
The count() is a built-in function in Python. It will return you the count of a given element in the list.

Algorithm:
Step-1: Define function list_count_6(nums).
Step-2: Set variable count to 0.
Step-3: For each num in nums, do the following:
If num is equal to 6, add 1 to count.
Step-4: Return count.
Step-5: Call list_count_6() with a sample list and print the result.



Disclaimer : We recommend you to try the code by yourself before checking the solution.

def list_count_6(nums):
 count = 0
 for num in nums:
  if num == 6:
  count = count + 1
 return count
print(list_count_6([1, 4, 6, 7, 4,6]))
