Q11(a). 2’s complement of a number is obtained by scanning it from right to left and complementing all the bits after the first appearance of a 1. Thus 2’s complement of 11100 is 00100. Write a C program to find the 2’s complement of a binary number.

Aim: To write a C program to find the 2's complement of a binary number by scanning it from right to left and complementing all the bits after the first appearance of a 1. The program should take the binary number as input from the user and output the 2's complement representation of the number. The aim of this experiment is to demonstrate the implementation of the 2's complement conversion process in the C programming language.

System and Software Requirements: A Computer with any Operating system with GCC compiler installed in it.

Description:
In binary arithmetic, the 2's complement of a binary number is used to represent negative numbers. The 2's complement of a binary number is obtained by scanning it from right to left and complementing all the bits after the first appearance of a 1. The resulting binary number represents the negative version of the original number.
To find the 2's complement of a binary number, the following steps are taken:
Complement the binary number by changing all 0s to 1s and all 1s to 0s.
Add 1 to the complemented number.
For example, the 2's complement of the binary number 11100 is obtained by first complementing it to 00011, and then adding 1 to it, resulting in 00100.
In the C programming language, bitwise operations can be used to implement the 2's complement conversion process. The program can take the binary number as input from the user, complement the number using bitwise operations, and add 1 to the result to find the 2's complement representation of the number. The final result can be displayed as output to the user.



Examples:
For Example: consider the binary number 1001. To find its 2's complement, we first take its complement, which is 0110. Then, we add 1 to the result, which gives us 0111. This is the 2's complement representation of the binary number 1001.

#include<stdio.h>
#include<string.h>
void twos_complement(char binary[]) {
int n = strlen(binary);
int i;
for (i = n - 1; i >= 0; i--) {
if (binary[i] == '0') {
binary[i] = '1';
} else {
binary[i] = '0';
}
}
for (i = n - 1; i >= 0; i--) {
if (binary[i] == '0') {
binary[i] = '1';
break;
} else {
binary[i] = '0';
}
}
}
int main() {
char binary[100];
printf("Enter a binary number: ");
scanf("%s", binary);
twos_complement(binary);
printf("The 2's complement of %s is %s\n", binary, binary);
return 0;
}
