Q5(a). Write C program that reads two integers x and n and calls a recursive function to compute xn.

Aim: To write a C program that computes xn using a recursive function.

System and Software Requirements: A Computer with any Operating system with GCC compiler installed in it.

Description:
The theory behind the program that reads two integers x and n and calls a recursive function to compute xn is based on the mathematical concept of exponentiation. The exponentiation operation, denoted as x^n, computes the value of x raised to the power of n. For example, if x = 2 and n = 3, then x^n = 2^3 = 8.
In the program, the function to compute xn is implemented using a recursive approach. The function takes two arguments, x and n, and returns x multiplied by itself n times. The recursion terminates when n becomes 0, at which point the function returns 1.
For example, if x = 2 and n = 3, the function would perform the following calculations:
x^3 = x * x^(3-1)
= x * x^2
= x * (x * x^(2-1))
= x * (x * x^1)
= x * (x * x)
= x * x * x
= 2 * 2 * 2
= 8
This recursive approach allows the computation of xn to be performed efficiently, with the number of calculations required being proportional to the logarithm of n.



Examples:
For Example: if x = 5 and n = 4, the function would perform the following operations:
x^4 = 5 * x^(4-1) = 5 * x^3
x^3 = 5 * x^(3-1) = 5 * x^2
x^2 = 5 * x^(2-1) = 5 * x^1
x^1 = 5 * x^(1-1) = 5 * x^0
x^0 = 1
Finally, the result is computed as:
x^4 = 5 * x^3 = 5 * (5 * x^2)
= 5 * (5 * (5 * x^1))
= 5 * (5 * (5 * (5 * x^0)))
= 5 * (5 * (5 * (5 * 1)))
= 5 * 5 * 5 * 5
= 625.
Thus, the recursive function successfully computes the value of x^n = 5^4 = 625.

#include<stdio.h>
int power(int x, int n)
{
if (n == 0)
return 1;
else
return x * power(x, n-1);
}
int main()
{
int x, n, result;
printf("Enter x and n: ");
scanf("%d %d", &x, &n);
result = power(x, n);
printf("%d^%d = %d", x, n, result);
return 0;
}
