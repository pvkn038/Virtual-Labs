C PROGRAMMING FOR PROBLEM SOLVING LAB

Q7(a). Write a menu-driven C program that allows a user to enter n numbers and then choose between finding the smallest, largest, sum, or average. The menu and all the choices are to be functions. Use a switch statement to determine what action to take. Display an error message if an invalid choice is entered.

Aim: To implement a menu-driven C program that allows a user to enter n numbers and then choose between finding the smallest, largest, sum, or average. The menu and all the choices are to be functions. Use a switch statement to determine what action to take. Display an error message if an invalid choice is entered.

System and Software Requirements: A Computer with any Operating system with GCC compiler installed in it.

Description:
A menu-driven program is a type of user interface that presents a list of options for the user to choose from. In this specific case, the program allows a user to enter 'n' numbers and then select an option from the menu, which consists of finding the smallest number, largest number, sum of all numbers, or the average of all numbers.
Each option from the menu is implemented as a separate function. When the user selects an option, the corresponding function is executed to perform the desired calculation. The implementation of this program uses a switch statement, which evaluates the user's choice and executes the corresponding function accordingly.
For instance, if the user selects the option to find the smallest number, the program will call the function that performs this calculation. The function will then search through the list of 'n' numbers entered by the user and return the smallest number. The same process applies to the other options in the menu.
If an invalid choice is entered, the program will display an error message to inform the user that the choice is not valid. This helps to ensure that the program behaves as expected and that the user is guided to make a valid selection.
In summary, the menu-driven program allows the user to enter 'n' numbers and then perform various calculations such as finding the smallest number, largest number, sum of all numbers, or average of all numbers, by selecting an option from the menu and executing the corresponding function. The program uses a switch statement to determine the action to be taken and displays an error message if an invalid choice is entered.
Let's consider an example where the user enters 5 numbers: 2, 4, 6, 8, and 10. The menu-driven program then presents the user with the following options:
Find the smallest number
Find the largest number
Find the sum of all numbers
Find the average of all numbers
If the user selects option 1 (Find the smallest number), the program will call the function responsible for finding the smallest number. This function will loop through the list of 5 numbers and compare each number to find the smallest. In this case, the smallest number is 2. The function will then return 2 as the result.
If the user selects option 2 (Find the largest number), the program will call the function responsible for finding the largest number. This function will loop through the list of 5 numbers and compare each number to find the largest. In this case, the largest number is 10. The function will then return 10 as the result.
If the user selects option 3 (Find the sum of all numbers), the program will call the function responsible for finding the sum of all numbers. This function will loop through the list of 5 numbers and add them together. In this case, the sum of the numbers is 30. The function will then return 30 as the result.
If the user selects option 4 (Find the average of all numbers), the program will call the function responsible for finding the average of all numbers. This function will first find the sum of all numbers and then divide it by the total number of elements (5 in this case). In this case, the average of the numbers is 6. The function will then return 6 as the result.
If the user enters an invalid choice, the program will display an error message such as "Invalid Choice. Please try again." This message helps the user understand that their selection is not valid and guides them to make a valid selection.

#include<stdio.h>
// Function to find the smallest number
int findSmallest(int numbers[], int n) {
int smallest = numbers[0];
for (int i = 1; i < n; i++) {
if (numbers[i] < smallest) {
smallest = numbers[i];
}
}
return smallest;
}
// Function to find the largest number
int findLargest(int numbers[], int n) {
int largest = numbers[0];
for (int i = 1; i < n; i++) {
if (numbers[i] > largest) {
largest = numbers[i];
}
}
return largest;
}
// Function to find the sum of all numbers
int findSum(int numbers[], int n) {
int sum = 0;
for (int i = 0; i < n; i++) {
sum += numbers[i];
}
return sum;
}
// Function to find the average of all numbers
float findAverage(int numbers[], int n) {
int sum = findSum(numbers, n);
float average = (float) sum / n;
return average;
}
// Function to display the menu and return the user's choice
int displayMenu() {
int choice;
printf("\nEnter your choice:\n");
printf("1. Find the smallest number\n");
printf("2. Find the largest number\n");
printf("3. Find the sum of all numbers\n");
printf("4. Find the average of all numbers\n");
printf("5. Quit\n");
scanf("%d", &choice);
return choice;
}
int main() {
int n;
printf("Enter the number of elements: ");
scanf("%d", &n);
int numbers[n];
printf("Enter %d numbers:\n", n);
for (int i = 0; i < n; i++) {
scanf("%d", &numbers[i]);
}
int choice = displayMenu();
while (choice != 5) {
switch (choice) {
case 1:
printf("The smallest number is: %d\n", findSmallest(numbers, n));
break; case 2: printf("The largest number is: %d\n", findLargest(numbers, n));
break;
case 3:
printf("The sum of all numbers is: %d\n", findSum(numbers, n));
break;
case 4:
printf("The average of all numbers is: %.2f\n", findAverage(numbers, n));
break;
default:
printf("Invalid choice. Please try again.\n");
}
choice = displayMenu();
}
return 0;
}
