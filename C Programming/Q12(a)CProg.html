Q12(a). Write a C program to display the contents of a file to standard output device.

Aim: To implement a C program to display the contents of a file to standard output device.

System and Software Requirements: A Computer with any Operating system with GCC compiler installed in it.

Description:
In C programming, the standard output device is referred to as "stdout". To display the contents of a file on the stdout, we need to perform the following steps:
1. Open the file: Using the fopen() function, we can open a file for reading. The fopen() function returns a pointer to a FILE type, which is used for subsequent file operations.
2. Read the file: Using the fread() function, we can read the contents of the file and store them in a buffer. The fread() function takes three arguments: the buffer, the size of each element to be read, and the number of elements to be read.
3. Write to file: Using the fwrite() function, we can write the contents of the buffer to the stdout. The fwrite() function takes four arguments: the buffer, the size of each element, the number of elements, and the FILE pointer.
4. Close the file: Finally, we need to close the file using the fclose() function to free up resources and prevent data corruption.



Examples:
For Example: To display the contents of a file to stdout:
1. Create a text file named sample.txt and write some content in it.
$ echo "Hello, this is a sample text file" > sample.txt
2. Compile and run the program, passing the file name as an argument:
$ gcc -o display display.c
$ ./display sample.txt
Hello, this is a sample text file
As you can see, the contents of the file sample.txt are displayed on the screen. The program reads the contents of the file, stores it in a buffer, and writes the buffer to the standard output device (stdout).

#include<stdio.h>
#include<stdlib.h>
int main(int argc, char *argv[]) {
FILE *file;
char *buffer;
unsigned long fileLen;
// Open file
file = fopen(argv[1], "rb");
if (!file) {
fprintf(stderr, "Unable to open file %s", argv[1]);
return 1;
}
// Get file length
fseek(file, 0, SEEK_END);
fileLen=ftell(file);
fseek(file, 0, SEEK_SET);
// Allocate memory
buffer=(char *)malloc(fileLen+1);
if (!buffer) {
fprintf(stderr, "Memory error!");
fclose(file);
return 1;
}
// Read file contents into buffer
fread(buffer, fileLen, 1, file);
// Write buffer to stdout
fwrite(buffer, fileLen, 1, stdout);
// Close file and free memory
fclose(file);
free(buffer);
return 0;
}

#include<stdio.h>
int main(int argc, char *argv[]) {
FILE *fp;
int c;
if (argc != 2) {
printf("Usage: %s filename\n", argv[0]);
return 1;
}
fp = fopen(argv[1], "r");
if (fp == NULL) {
printf("Error opening file %s\n", argv[1]);
return 1;
}
while ((c = fgetc(fp)) != EOF) {
putchar(c);
}
fclose(fp);
return 0;
}
