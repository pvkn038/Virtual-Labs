Q9(a). Write a C program that uses functions to perform the following operations:
i) To insert a sub-string into a given main string from a given position.
ii) To delete n characters from a given position in a given string.

Aim: To design a C program that utilizes functions to perform the operations of inserting a sub-string into a given main string from a specific position and to delete n characters from a given position in a given string. The program will demonstrate the ability to manipulate strings in C programming by implementing the aforementioned operations.

System and Software Requirements: A Computer with any Operating system with GCC compiler installed in it.

Description:
To perform the operations described above, we will create two functions in C: one to insert a sub-string into a main string and one to delete characters from a string. These functions will take the following parameters:
Insert function:

  • A character array (string) to store the main string
  • An integer to represent the position where the sub-string should be inserted
  • A character array (sub-string) to be inserted into the main string.
Delete function:

  • A character array (string) to store the main string
  • An integer to represent the position from where characters should be deleted
  • An integer to represent the number of characters to be deleted.
The insert function will start by allocating enough memory to store the main string, the sub-string and the inserted string. This can be done using the malloc function. Next, the sub-string will be inserted into the main string by using a loop to copy the characters from the main string and the sub-string into the new string. The loop will start at the specified position and copy characters from the main string until the end of the string, then it will copy characters from the sub-string until the end of the sub-string. Finally, the new string will be returned as the result of the function.
The delete function will start by allocating enough memory to store the main string and the modified string. This can be done using the malloc function. Next, the characters to be deleted will be skipped over by using a loop to copy the characters from the main string into the new string, starting at the specified position. The loop will continue until the end of the main string, and the new string will be returned as the result of the function.
The experimental theory will be tested by running the functions with different inputs and verifying that the correct output is produced. For example, if the main string is "Hello, world!" and the sub-string is "there", the insert function should return "Hello, there world!" if the position is 7. If the main string is "Hello, world!" and the position is 7 and the number of characters to delete is 6, the delete function should return "Hello, world".
Let's consider the main string "Hello", the sub-string "there" and the position 2. The program will insert the sub-string into the main string as shown below:
"Hetherello"
Delete n characters from a given position in a given string needs to create a function that takes three inputs, the main string, the position, and the number of characters to be deleted. The program then uses string manipulation functions to shift the characters in the main string after the given position, n spaces to the left, effectively deleting the characters.





Examples:
For Example: Let's consider the main string "Hello", the position 2, and the number of characters to be deleted as 2. The program will delete the characters from the main string as shown below:
"Heo"

i) To insert a sub-string into a given main string from a given position.
#include<stdio.h>
#include<string.h>
void insert_substring(char main_string[], char substring[], int position) {
int main_string_length = strlen(main_string);
int substring_length = strlen(substring);
// Shift the characters in the main string to the right to make room for the substring
for (int i = main_string_length - 1; i >= position; i--) {
main_string[i + substring_length] = main_string[i];
}
// Insert the substring into the main string
for (int i = 0; i < substring_length; i++) {
main_string[position + i] = substring[i];
}
}
int main() {
char main_string[100] = "Hello, World!";
char substring[100] = "beautiful ";
int position = 7;
printf("Original main string: %s\n", main_string);
insert_substring(main_string, substring, position);
printf("Modified main string: %s\n", main_string);
return 0;
}

ii) To delete n characters from a given position in a given string.
#include<stdio.h>
#include<string.h>
#define MAX_LEN 100
void delete_characters(char str[], int position, int n) {
int len = strlen(str);
if (position + n > len) {
printf("Error: Cannot delete more characters than present in the string.\n");
return;
}
for (int i = position; i <= len - n; i++) {
str[i] = str[i + n];
}
str[len - n] = '\0';
}
int main() {
char str[MAX_LEN];
int position, n;
printf("Enter a string: ");
scanf("%s", str);
printf("Enter the position from where characters are to be deleted: ");
scanf("%d", &position);
printf("Enter the number of characters to be deleted: ");
scanf("%d", &n);
delete_characters(str, position, n);
printf("The modified string is: %s\n", str);
return 0;
}
