Q13(b). Write a C program to compare two files, printing the first line where they differ.

Aim: To implement a C program to compare two files, printing the first line where they differ.

System and Software Requirements: A Computer with any Operating system with GCC compiler installed in it.

Description:
This experiment aims to compare two files and print the first line where they differ. The comparison will be performed using mathematical examples. The mathematical examples will consist of two sets of numbers, and the experiment will compare each number in the sets to determine if they are the same or different.
Methods:
1. Open the two text files in a text editor and examine the contents to ensure that they contain sets of numbers.
2. Write a program in Python that will compare each number in the two sets and determine if they are the same or different.
3. The program will compare each line in the two files and print the first line where they differ.
The expected result of this experiment is that the program will compare the two sets of numbers and print the first line where they differ. The line of difference will be the first line in which a number in one set is different from the corresponding number in the other set.
The experiment will demonstrate how to compare two files and print the first line where they differ. The use of mathematical examples will allow for a clear understanding of the concept of file comparison and the output of the program. This experiment can be extended to compare larger files or different types of data, such as strings or lists, to further explore the capabilities of file comparison.
Given two sets of numbers, A and B, the goal is to determine if they are the same or different. The sets can be represented as arrays, A = [a1, a2, a3, ..., an] and B = [b1, b2, b3, ..., bn].
To compare the sets, we can iterate through each element in the arrays and compare the corresponding elements. If a difference is found, we can return the index of the difference. If no differences are found, the sets are considered to be the same.



Examples:
For Example: Let's consider an example:
A = [1, 2, 3, 4, 5]
B = [1, 2, 4, 5, 6]
Iterating through the arrays, we can compare the corresponding elements:
a1 and b1 are the same (1 == 1)
a2 and b2 are the same (2 == 2)
a3 and b3 are different (3 != 4)
Since a difference has been found, we can return the index of the difference, which is 2. This means that the first line where the sets differ is the third line.
This mathematical theory can be applied to comparing two files by representing each line in the files as an element in an array. The comparison can be performed on a line-by-line basis, with the result being the first line where the sets differ.

#include<stdio.h>
#include<stdlib.h>
#include<string.h>
#define MAX_LINE_LENGTH 1000
int main(int argc, char *argv[]) {
if (argc != 3) {
printf("Usage: %s \n", argv[0]);
return 1;
}
FILE *file1 = fopen(argv[1], "r");
FILE *file2 = fopen(argv[2], "r");
if (!file1 || !file2) {
printf("Error: unable to open files\n");
return 1;
}
char line1[MAX_LINE_LENGTH];
char line2[MAX_LINE_LENGTH];
int line_number = 1;
while (fgets(line1, MAX_LINE_LENGTH, file1) &&
fgets(line2, MAX_LINE_LENGTH, file2)) {
if (strcmp(line1, line2) != 0) {
printf("First line of difference: %d\n", line_number);
break;
}
line_number++;
}
fclose(file1);
fclose(file2);
return 0;
}

#include<stdio.h>
#include<stdlib.h>
#include<string.h>
#define MAX_LINE_LENGTH 1000
int compare_files(FILE *file1, FILE *file2) {
char line1[MAX_LINE_LENGTH];
char line2[MAX_LINE_LENGTH];
int line_number = 1;
while (fgets(line1, MAX_LINE_LENGTH, file1) &&
fgets(line2, MAX_LINE_LENGTH, file2)) {
if (strcmp(line1, line2) != 0) {
return line_number;
}
line_number++;
}
return 0;
}
int main(int argc, char *argv[]) {
if (argc != 3) {
printf("Usage: %s \n", argv[0]);
return 1;
}
FILE *file1 = fopen(argv[1], "r");
FILE *file2 = fopen(argv[2], "r");
if (!file1 || !file2) {
printf("Error: unable to open files\n");
return 1;
}
int line_number = compare_files(file1, file2);
if (line_number) {
printf("First line of difference: %d\n", line_number);
} else {
printf("Files are identical\n");
}
fclose(file1);
fclose(file2);
return 0;
}
