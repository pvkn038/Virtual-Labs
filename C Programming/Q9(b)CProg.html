Q9(b). Write a C program that uses a non-recursive function to determine if the given string is a palindrome or not.

Aim: To implement a C program that uses a non-recursive function to determine if the given string is a palindrome or not.

System and Software Requirements: A Computer with any Operating system with GCC compiler installed in it.

Description:
Palindromes are words or phrases that read the same backward as they do forward. A non-recursive function can be used to determine if a given string is a palindrome or not by comparing the first and last characters of the string and then moving inward until the middle of the string is reached. For Example: Consider the string "racecar". To determine if this string is a palindrome, the first and last characters are compared. If they match, then the next pair of characters is compared, and so on, until either a mismatch is found or the middle of the string is reached.
Compare the first and last characters: "r" and "r" match.
Compare the next pair of characters: "a" and "a" match.
Compare the next pair of characters: "c" and "c" match.
Compare the next pair of characters: "e" and "e" match.
Compare the next pair of characters: "c" and "c" match.
The middle of the string has been reached, and all pairs of characters have matched. Therefore, the string "racecar" is a palindrome.
In C program, this can be implemented as a loop that iterates through the string and compares the characters at each end of the string, moving inward until the middle is reached or a mismatch is found. The function can then return a value indicating whether the string is a palindrome or not.



Examples:
For Example: Consider the string "hello". To determine if this string is a palindrome, the first and last characters are compared. If they match, then the next pair of characters is compared, and so on, until either a mismatch is found or the middle of the string is reached.
Compare the first and last characters: "h" and "o" do not match.
The middle of the string has not been reached, but a mismatch has been found. Therefore, the string "hello" is not a palindrome.
In C program, this can be implemented as a loop that iterates through the string and compares the characters at each end of the string, moving inward until the middle is reached or a mismatch is found. If a mismatch is found before the middle of the string is reached, the function can return a value indicating that the string is not a palindrome.

#include<stdio.h>
#include<string.h>
int is_palindrome(char *str) {
int i, j;
int len = strlen(str);
for (i = 0, j = len - 1; i < j; i++, j--) {
if (str[i] != str[j])
return 0;
}
return 1;
}
int main() {
char str[100];
printf("Enter a string: ");
scanf("%s", str);
if (is_palindrome(str))
printf("The string is a palindrome\n");
else
printf("The string is not a palindrome\n");
return 0;
}
