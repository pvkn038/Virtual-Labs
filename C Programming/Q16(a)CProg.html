Q16(a). Write a C Program to calculate the sum of n numbers entered by the user using malloc( ) and free() functions.

Aim: To implement a C Program to calculate the sum of n numbers entered by the user using malloc( ) and free() functions.

System and Software Requirements: A Computer with any Operating system with GCC compiler installed in it.

Description:
The malloc() function in C is used to dynamically allocate memory during runtime. It takes the size of the memory block to be allocated as an argument and returns a pointer to the first byte of the allocated memory. The free() function is used to deallocate the dynamically allocated memory.
Procedure:
1. Initialize two pointer variables 'ptr1' and 'ptr2' to store the addresses of the first and second numbers, respectively.
2. Allocate memory for the first number using the malloc() function, storing the address of the allocated memory block in the 'ptr1' variable.
3. Ask the user to enter the first number, and store it in the dynamically allocated memory block.
4. Allocate memory for the second number using the malloc() function, storing the address of the allocated memory block in the 'ptr2' variable.
5. Ask the user to enter the second number, and store it in the dynamically allocated memory block.
6. Initialize a variable 'sum' to store the sum of the two numbers.
7. Add up the two numbers and store the result in the 'sum' variable.
8. Print the value of 'sum' to the screen.
9. Deallocate the dynamically allocated memory blocks using the free() function.
The program should correctly calculate the sum of the two numbers entered by the user and print the result on the screen. Additionally, the program should correctly deallocate the dynamically allocated memory using the free() function, avoiding any memory leaks.
This experiment demonstrates the use of dynamic memory allocation techniques in C programming using the malloc() and free() functions. The program correctly calculates the sum of two numbers entered by the user and deallocates the dynamically allocated memory, making it a useful and efficient technique for memory management in C programming.



Examples:
For Example: Suppose we want to find the sum of two numbers, a and b. We can use malloc() to dynamically allocate memory for the two numbers and then use free() to deallocate the memory after we are done with it.
Here's how it would work in theory:
1. Call the malloc() function to allocate memory for the first number, a. This function takes the size of the memory block needed (in this case, the size of an int data type) as an argument and returns a pointer to the first byte of the allocated memory.
2. Store the value of the first number, a, in the allocated memory block.
3. Call the malloc() function again to allocate memory for the second number, b.
4. Store the value of the second number, b, in the allocated memory block.
5. Calculate the sum of the two numbers by adding a and b.
6. Print the sum.
7. Call the free() function to deallocate the memory for the two numbers. This function takes the pointer to the memory block as an argument and releases it, making it available for other parts of the program to use.
In this example, we used malloc() to dynamically allocate memory for the two numbers, a and b, and free() to deallocate the memory after we were done with it. By using dynamic memory allocation, we can avoid memory leaks and improve the efficiency of our program.

#include<stdio.h>
#include<stdlib.h>
int main() {
int n, i, sum = 0;
int *ptr;
printf("Enter the number of elements: ");
scanf("%d", &n);
ptr = (int *) malloc(n * sizeof(int)); // Allocating memory for n integers
if (ptr == NULL) {
printf("Memory allocation failed.\n");
exit(0);
}
printf("Enter %d numbers: \n", n);
for (i = 0; i < n; i++) {
scanf("%d", ptr + i); // Reading n numbers
sum += *(ptr + i); // Calculating the sum
}
printf("Sum of the numbers = %d\n", sum);
free(ptr); // Deallocating the memory
return 0;
}

for (i = 0; i < n; i++) {
printf("Enter number %d: ", i + 1);
