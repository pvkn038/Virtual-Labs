Q13(a). Write a C program to count the number of times a character occurs in a text file. The file name and the character are supplied as command-line arguments.

Aim: To implement a C program to count the number of times a character occurs in a text file. The file name and the character are supplied as command-line arguments.

System and Software Requirements: A Computer with any Operating system with GCC compiler installed in it.

Description:
In computer science, the process of counting the number of occurrences of a specific character in a text file can be performed using various algorithms. The simplest method is to read the text file character by character and increment a counter each time the specified character is found.
To implement this in C programming, we can use file handling functions provided by the C standard library, such as fopen(), fgetc(), and fclose(). The fopen() function is used to open the specified text file, while the fgetc() function is used to read individual characters from the file. The fclose() function is used to close the file after processing is complete.
In addition to file handling functions, we also need to use command-line arguments to supply the file name and character to the program. Command-line arguments are supplied to the main function of the program and can be accessed through the argv[] array.
In summary, the experimental theory for the C program to count the number of occurrences of a character in a text file involves using file handling functions, command-line arguments, and a loop to read the text file character by character and increment a counter each time the specified character is found. The final result will be the number of times the character appears in the file, which can be displayed to the user.



Examples:
For Example: Consider the following text file named "sample.txt" with the following contents:
hello world
If we want to count the number of occurrences of the letter 'l' in the file, the program would perform the following steps:
Open the file using the fopen() function:
FILE *fp;
fp = fopen("sample.txt", "r");
Read the file character by character using the fgetc() function and increment a counter each time the character 'l' is found:
int count = 0;
char ch;
while((ch = fgetc(fp)) != EOF) {
if (ch == 'l') {
count++;
}
}
Close the file using the fclose() function:
fclose(fp);
The final result, which is the number of occurrences of the letter 'l', can be displayed to the user:
perl
printf("The letter 'l' occurs %d times in the file.\n", count);
So, in this example, the letter 'l' occurs 3 times in the file "sample.txt".

#include<stdio.h>
#include<string.h>
int main(int argc, char *argv[]) {
if (argc != 3) {
printf("Usage: %s \n", argv[0]);
return 1;
}
FILE *fp;
fp = fopen(argv[1], "r");
if (fp == NULL) {
printf("Error opening file %s\n", argv[1]);
return 1;
}
char contents[100];
fgets(contents, 100, fp);
int count = 0;
char *p = contents;
while (p = strchr(p, *argv[2])) {
count++;
p++;
}
fclose(fp);
printf("The character '%c' occurs %d times in the file %s.\n", *argv[2], count, argv[1]);
return 0;
}
