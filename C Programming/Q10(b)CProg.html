C PROGRAMMING FOR PROBLEM SOLVING LAB

Q10(b). Write a C program that reads 15 names each of up to 30 characters, stores them in an array, and uses an array of pointers to display them in ascending (ie. alphabetical) order.

Aim: To implement a C program that reads 15 names each of up to 30 characters, stores them in an array, and uses an array of pointers to display them in ascending (ie. alphabetical) order.

System and Software Requirements: A Computer with any Operating system with GCC compiler installed in it.

Description:
This C program demonstrates the use of arrays and pointers in organizing and displaying data. The program first prompts the user to input 15 names, each of up to 30 characters. These names are stored in a single-dimensional array, for example, let's say the array is named "names". The program then uses an array of pointers, with each element pointing to a name stored in the "names" array. This array of pointers is named "pointers", where "pointers[i]" points to "names[i]".
To sort the names in ascending order, the program uses a sorting algorithm such as Bubble Sort. The algorithm compares the ASCII values of the names pointed to by consecutive elements in the "pointers" array. If the ASCII value of "pointers[i]" is greater than "pointers[i+1]", the two elements are swapped. This process continues until the "pointers" array is sorted in ascending order.
Mathematically, let's say the "names" array has elements "A", "C", "B", and "pointers" array has elements "p1", "p2", "p3". Initially, "p1" points to "A", "p2" points to "C", and "p3" points to "B". After sorting, "p1" points to "A", "p2" points to "B", and "p3" points to "C".
The sorted "pointers" array is then used to display the names in the desired order. This program highlights the use of pointers as a means of organizing and manipulating data, as well as the sorting algorithms used to arrange data in a specific order.



Examples:
For Example: let's say the "names" array has elements "John", "Alice", "Bob", and the "pointers" array has elements "p1", "p2", "p3". Initially, "p1" points to "John", "p2" points to "Alice", and "p3" points to "Bob". After sorting, "p1" points to "Alice", "p2" points to "Bob", and "p3" points to "John".
The sorted "pointers" array is then used to display the names in the desired order. This program highlights the use of pointers as a means of organizing and manipulating data, as well as the sorting algorithms used to arrange data in a specific order.

#include<stdio.h>
#include<string.h>
#define MAX_NAMES 15
#define MAX_LENGTH 30
int main() {
char names[MAX_NAMES][MAX_LENGTH]; // array to store names
char *pointers[MAX_NAMES]; // array of pointers to store addresses of names
// Prompt user to input names
printf("Enter %d names, each up to %d characters:\n", MAX_NAMES, MAX_LENGTH - 1);
for (int i = 0; i < MAX_NAMES; i++) {
scanf("%s", names[i]);
pointers[i] = names[i];
}
// Sort names in ascending order using pointers array
char *temp;
for (int i = 0; i < MAX_NAMES - 1; i++) {
for (int j = 0; j < MAX_NAMES - i - 1; j++) {
if (strcmp(pointers[j], pointers[j + 1]) > 0) {
temp = pointers[j];
pointers[j] = pointers[j + 1];
pointers[j + 1] = temp;
}
}
}
// Display names in ascending order
printf("\nNames in ascending order:\n");
for (int i = 0; i < MAX_NAMES; i++) {
printf("%s\n", pointers[i]);
}
return 0;
}
