Q10(a). Write a C program to replace a substring with another in a given line of text.

Aim: To implement a C program to replace a substring with another in a given line of text.

System and Software Requirements: A Computer with any Operating system with GCC compiler installed in it.

Description:
A substring can be replaced with another in a given line of text by first finding the position of the substring to be replaced, then creating a new string that includes the text before the substring, the replacement substring, and the text after the substring. For example: Consider the line of text "The cat is sleeping" and the substrings "cat" and "dog". To replace "cat" with "dog", the position of "cat" must first be found in the text.
Find the position of "cat": "The cat is sleeping" starts at position 4.
Create a new string by combining the text before "cat", the replacement substring "dog", and the text after "cat": "The dog is sleeping".
In C program, this can be implemented using string manipulation functions such as strstr() to find the position of the substring to be replaced, strncpy() to copy the text before the substring, strcpy() to copy the replacement substring, and strcat() to concatenate the text after the substring to the new string. The program can loop through the line of text to find and replace all occurrences of the substring.



Examples:
For Example: Consider the line of text "I love to play soccer" and the substrings "love" and "hate". To replace "love" with "hate", the position of "love" must first be found in the text.
Find the position of "love": "I love to play soccer" starts at position 2.
Create a new string by combining the text before "love", the replacement substring "hate", and the text after "love": "I hate to play soccer".
In C program, this process can be repeated for each occurrence of the substring to be replaced, updating the new string each time.

#include<stdio.h>
#include<string.h>
#include<stdlib.h>
char* replace_substring(const char* str, const char* sub, const char* replace) {
char *result;
int count = 0;
int length, sub_len, replace_len;
const char *temp;
length = strlen(str);
sub_len = strlen(sub);
replace_len = strlen(replace);
/* Count the number of times the substring occurs in the string */
temp = str;
while ((temp = strstr(temp, sub))) {
count++;
temp += sub_len;
}
/* Allocate memory for the new string */
result = malloc(length + (replace_len - sub_len) * count + 1);
if (!result) return NULL;
/* Replace the substring with the replacement in the new string */
temp = str;
char *ptr = result;
while ((temp = strstr(temp, sub))) {
length = temp - str;
strncpy(ptr, str, length);
ptr += length;
strcpy(ptr, replace);
ptr += replace_len;
temp += sub_len;
str = temp;
}
strcpy(ptr, str);
return result;
}
int main() {
char line[] = "The cat is sleeping";
char sub[] = "cat";
char replace[] = "dog";
char *new_line = replace_substring(line, sub, replace);
printf("Original: %s\n", line);
printf("Modified: %s\n", new_line);
free(new_line);
return 0;
}
