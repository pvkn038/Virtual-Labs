Q14(a). Write a C program to change the nth character (byte) in a text file. Use fseek function.

Aim: To implement a C program to change the nth character (byte) in a text file. Use fseek function.

System and Software Requirements: A Computer with any Operating system with GCC compiler installed in it.

Description:
The experimental theory behind changing the nth character (byte) in a text file using the fseek function is based on the concept of file pointers in C programming. In C, a file pointer is a variable that holds the memory address of the first byte of a file. When we open a file using the fopen function, it returns a pointer to the first byte of the file. This pointer can be used to access and manipulate the contents of the file.
The fseek function is used to move the file pointer to a specific position in the file. It takes three arguments: the file pointer, the number of bytes to move the file pointer, and the direction in which to move the file pointer. By using the fseek function, we can move the file pointer to the nth character (byte) in the text file and change its value.
To change the nth character (byte), we first need to open the text file in "read/write" mode using the fopen function. Then, we use the fseek function to move the file pointer to the nth character (byte). Finally, we use the fwrite function to write a new character at the current position of the file pointer.
After making the changes, we can validate the changes by reading and printing the contents of the text file using the fread function. The contents of the text file should show that the nth character (byte) has been successfully changed.



Examples:
For Example: Consider a text file with the following contents:
Hello World
Let's say we want to change the 6th character (byte) from 'W' to 'X'.
In mathematical terms, we can represent the contents of the text file as a string s of length n:
s = "Hello World"
n = 11
The file pointer is initially pointing to the first character (byte) of the string, i.e. s[0].
Using the fseek function, we can move the file pointer to the 6th character (byte) of the string, i.e. s[5]:
fseek(file_pointer, 5, SEEK_SET) Here, file_pointer is the file pointer obtained from the fopen function, 5 is the number of bytes to move the file pointer, and SEEK_SET is the direction to move the file pointer (in this case, from the beginning of the file).
Next, we use the fwrite function to write a new character at the current position of the file pointer:
fwrite("X", 1, 1, file_pointer)
Here, "X" is the new character to write, 1 is the size of each element to write, and 1 is the number of elements to write.
After making these changes, the contents of the text file will be:
Hello Xorld
We can validate the changes by reading and printing the contents of the text file using the fread function. The output should be Hello Xorld, showing that the 6th character (byte) has been successfully changed from 'W' to 'X'.

#include<stdio.h>
int main() {
FILE *file_pointer;
char old_string[] = "Hello World";
int n = 11; // length of the string
int position = 5; // position of the character to change
// Open the text file in "read/write" mode
file_pointer = fopen("file.txt", "r+");
if (file_pointer == NULL) {
printf("Error opening file!\n");
return 1;
}
// Write the original string to the text file
fwrite(old_string, 1, n, file_pointer);
// Move the file pointer to the nth character (byte)
fseek(file_pointer, position, SEEK_SET);
// Replace the nth character (byte) with a new character
fwrite("X", 1, 1, file_pointer);
// Close the text file
fclose(file_pointer);
printf("Successfully changed the %dth character in the text file.\n", position + 1);
return 0;
}

#include<stdio.h>
#include<string.h>
int main() {
FILE *file_pointer;
char old_string[] = "Hello World";
int n = 11; // length of the string
int position = 5; // position of the character to change
char new_char = 'X';
// Open the text file in "write" mode
file_pointer = fopen("file.txt", "w");
if (file_pointer == NULL) {
printf("Error opening file!\n");
return 1;
}
// Write the original string to the text file
fwrite(old_string, 1, n, file_pointer);
// Close the text file
fclose(file_pointer);
// Open the text file in "read/write" mode
file_pointer = fopen("file.txt", "r+");
if (file_pointer == NULL) {
printf("Error opening file!\n");
return 1;
}
// Move the file pointer to the nth character (byte)
fseek(file_pointer, position, SEEK_SET);
// Replace the nth character (byte) with a new character
fwrite(&new_char, 1, 1, file_pointer);
// Close the text file
fclose(file_pointer);
printf("Successfully changed the %dth character in the text file.\n", position + 1);
return 0;
}
