Q4(a). The Least Common Multiple (LCM) of two positive integers a & b is the smallest integer that is evenly divisible by both a and b. Write a C program that reads two integers and calls GCD (a, b) function that takes two integer arguments and returns their LCM. The lcm (a, b) function should calculate the least common multiple by calling the GCD (a,b) function and using the following relation: LCM (a,b) = ab / GCD (a,b).

Aim: To write a C program that calculates the Least Common Multiple (LCM) of two positive integers a & b by using the GCD (a,b) function and the relation LCM (a,b) = ab / GCD (a,b).

System and Software Requirements: A Computer with any Operating system with GCC compiler installed in it.

Description:
The Least Common Multiple (LCM) of two positive integers, a and b, is the smallest positive integer that is evenly divisible by both a and b. To calculate the LCM, we can use the following relation: LCM (a,b) = ab / GCD (a,b), where GCD stands for the Greatest Common Divisor.
The GCD of two numbers is the largest number that divides both of them evenly. For example, the GCD of 12 and 16 is 4 because 4 is the largest number that can divide both 12 and 16 evenly.
Now, to find the LCM of 12 and 16, we can use the following calculation: LCM (12, 16) = 12 * 16 / GCD (12, 16) = 192 / 4 = 48. This means that 48 is the smallest number that is evenly divisible by both 12 and 16.
In other words, the LCM of two numbers is equal to the product of the two numbers divided by their GCD. By using this relation, we can calculate the LCM more efficiently by first finding the GCD and then using that result in the LCM calculation.


Examples:
Let's take two positive integers a=15 and b=20 as an example.
First, we need to find the GCD of 15 and 20. By using the Euclidean algorithm, we can find the GCD of 15 and 20 as follows:
GCD (15, 20) = GCD (15, 20 - 15) = GCD (15, 5) = 5
Now that we have the GCD of 15 and 20, we can use the following relation to find the LCM: LCM (15, 20) = 15 * 20 / GCD (15, 20) = 300 / 5 = 60.
So, the LCM of 15 and 20 is 60. This means that 60 is the smallest positive integer that is evenly divisible by both 15 and 20.

#include<stdio.h>
int gcd(int a, int b) {
int t;
while (b != 0) {
t = b;
b = a % b;
a = t;
}
return a;
}
int lcm(int a, int b) {
int result = a * b / gcd(a, b);
return result;
}
int main() {
int a, b;
printf("Enter two positive integers: ");
scanf("%d %d", &a, &b);
printf("The LCM of %d and %d is: %d\n", a, b, lcm(a, b));
return 0;
}
