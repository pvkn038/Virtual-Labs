Q6(b). Write a C program that uses non recursive function to search for a Key value in a given list of integers. Use linear search method.

Aim: To implement a C program using a non-recursive function to search for a key value in a given list of integers using the linear search method.

System and Software Requirements: A Computer with any Operating system with GCC compiler installed in it.

Description:
Linear search is a simple algorithm used to find the location of a key value within a list of integers. The linear search method works by iterating through the list of integers one by one, starting from the first element, until the key value is found or the end of the list is reached.
In this implementation, the linearSearch function takes three parameters: an array of integers arr, the number of elements in the array n, and the key value to be searched for key. The function returns the index of the key value within the array if it is found, and returns -1 if the key value is not found.
The main function initializes an array of integers arr, the number of elements in the array n, and the key value key. The linearSearch function is then called with the arr, n, and key as arguments, and the result is stored in a variable result. Finally, a message is printed to indicate whether the key value was found or not, and the index at which it was found if applicable.
Example: For the list of integers [4, 3, 6, 2, 8] and key value 6, the output of the program would be "Key found at index 2".
Note: The linear search algorithm has a time complexity of O(n) in the worst case, where n is the number of elements in the list. This means that the time taken by the algorithm increases linearly with the size of the list. For large lists, a more efficient search algorithm such as binary search should be used.


Examples:
For Example: Given a list of numbers [3, 5, 2, 7, 9], we want to find the target value 7.
Start at the first element in the list, 3.
Check if 3 is equal to 7, if not move to the next element.
Check if 5 is equal to 7, if not move to the next element.
Check if 2 is equal to 7, if not move to the next element.
Check if 7 is equal to 7, if so, the target value has been found.
Stop the search.
In this example, the target value 7 was found at index 3.

#include<stdio.h>
int linear_search(int arr[], int n, int key) {
int i;
for (i = 0; i < n; i++) {
if (arr[i] == key) {
return i;
}
}
return -1;
}
int main() {
int n;
printf("Enter the number of elements in the array: ");
scanf("%d", &n);
int arr[n];
printf("Enter the elements of the array: \n");
for (int i = 0; i < n; i++) {
scanf("%d", &arr[i]);
}
int key;
printf("Enter the key to be searched: ");
scanf("%d", &key);
int result = linear_search(arr, n, key);
if (result == -1) {
printf("Element is not present in the array\n");
} else {
printf("Element is present at index %d\n", result);
}
return 0;
}
