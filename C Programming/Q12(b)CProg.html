Q12(b).Write a C program which copies one file to another, replacing all lowercase characters with their uppercase equivalents.

Aim: To write a C program that copies the contents of one file to another, converting all lowercase characters to uppercase characters in the process.

System and Software Requirements: A Computer with any Operating system with GCC compiler installed in it.

Description:
This experiment aims to demonstrate the process of copying the contents of one file to another in the C programming language, while converting all lowercase characters to their uppercase equivalents using a simple mathematical approach.
To convert a lowercase character to its uppercase equivalent, we can use the following mathematical formula:
Uppercase = Lowercase - 32
where Lowercase is the ASCII value of the lowercase character and Uppercase is the ASCII value of its uppercase equivalent. The difference of 32 between the ASCII values of the two characters is due to the way that the ASCII table is organized, with uppercase characters having a higher ASCII value than their corresponding lowercase characters.
The experiment will make use of file input/output functions in C, including fopen(), fgetc(), and fputc(), to read and write the contents of the files. The program will first open the source file using fopen(), and use fgetc() to read its contents character by character. For each lowercase character read, the program will apply the above mathematical formula to convert it to its uppercase equivalent. The uppercase character will then be written to the destination file using fputc(). Finally, the program will close both files and output a message indicating that the conversion was successful.



Examples:
For Example: To convert a lowercase character to its uppercase equivalent, we can use the following mathematical formula:
Uppercase = Lowercase - ('a' - 'A')
where Lowercase is the ASCII value of the lowercase character and Uppercase is the ASCII value of its uppercase equivalent. The difference of 'a' - 'A' represents the difference between the ASCII values of the two characters, with 'A' having a higher ASCII value than 'a'.

#include<stdio.h>
#include<ctype.h>
int main() {
FILE *src, *dest;
char ch;
src = fopen("source.txt", "r");
dest = fopen("destination.txt", "w");
if (src == NULL || dest == NULL) {
printf("Error opening file\n");
return 1;
}
while ((ch = fgetc(src)) != EOF) {
if (islower(ch)) {
ch = toupper(ch);
}
fputc(ch, dest);
}
printf("File successfully copied and converted.\n");
fclose(src);
fclose(dest);
return 0;
}

#include<stdio.h>
#include<ctype.h>
int main(int argc, char *argv[]) {
// Check if the right number of arguments have been provided
if (argc != 3) {
printf("Usage: %s \n", argv[0]);
return 1;
}
// Open the input and output files
FILE *input_file = fopen(argv[1], "r");
FILE *output_file = fopen(argv[2], "w");
// Check if both files were successfully opened
if (!input_file) {
printf("Error: Unable to open input file %s\n", argv[1]);
return 1;
}
if (!output_file) {
printf("Error: Unable to open output file %s\n", argv[2]);
return 1;
}
// Read characters from the input file and write the uppercase equivalent to the output file
int c;
while ((c = fgetc(input_file)) != EOF) {
fputc(toupper(c), output_file);
}
// Close both files
fclose(input_file);
fclose(output_file);
return 0;
}
