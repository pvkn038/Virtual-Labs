C PROGRAMMING FOR PROBLEM SOLVING LAB

Q4(b). Write a C program that reads two integers n and r to compute the ncr value using the following relation: ncr (n,r) = n! / r! (n-r)!. Use a function for computing the factorial value of an integer.

Aim: To write a C program that computes the value of nCr(n, r) using the formula nCr(n, r) = n! / (r! (n-r)!) and to create a function that calculates the factorial value of an integer.

System and Software Requirements: A Computer with any Operating system with GCC compiler installed in it.

Description:
The number of combinations nCr of n items taken as r at a time is given by the relation nCr = n! / (r! (n-r)!), where n! is the factorial of n.
The factorial of a positive integer n is the product of all positive integers less than or equal to n. For example, the factorial of 5 is 5! = 5 x 4 x 3 x 2 x 1 = 120.
To implement the calculation of nCr in a program, we can create a function to compute the factorial of an integer, and then use that function to calculate nCr. The factorial function takes an integer n as an input and returns n! as the output.
In the main function, we first define n and r, and then call the nCr function to calculate the value of nCr(n, r). For example, if n is 5 and r is 2, the program will calculate nCr(5, 2) as 5! / (2! * (5-2)!), which is equal to 5! / (2! * 3!) or 5 x 4 / (2 x 1), which is equal to 10. The result is printed to the console.



Examples:
The value of nCr(n, r) can be calculated mathematically using the following formula:
nCr(n, r) = n! / (r! (n-r)!)
Where n! is the factorial of n and r! is the factorial of r.
For n = 23 and r = 2, the value of nCr(n, r) can be calculated as follows:
nCr(23, 2) = 23! / (2! (23-2)!)
= 23! / (2! 21!)
= 23 * 22 / 2
= 23 * 11
= 253
Therefore, nCr(23, 2) = 253.

#include<stdio.h>
int factorial(int num) {
int fact = 1;
for (int i = 1; i <= num; i++) {
fact = fact * i;
}
return fact;
}
int nCr(int n, int r) {
return factorial(n) / (factorial(r) * factorial(n - r));
}
int main() {
int n, r;
printf("Enter the value of n: ");
scanf("%d", &n);
printf("Enter the value of r: ");
scanf("%d", &r);
int result = nCr(n, r);
printf("The value of nCr(%d, %d) is: %d\n", n, r, result);
return 0;
}

#include<stdio.h>
// Function to compute the factorial of a given integer
int factorial(int num) {
int result = 1;
for (int i = 1; i <= num; i++) {
result *= i;
}
return result;
}
// Function to compute nCr
int nCr(int n, int r) {
return factorial(n) / (factorial(r) * factorial(n - r));
}
int main() {
int n, r;
printf("Enter values of n and r: ");
scanf("%d%d", &n, &r);
int nCrVal = nCr(n, r);
printf("nCr value of %d and %d is %d\n", n, r, nCrVal);
return 0;
}
