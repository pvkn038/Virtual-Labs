Q5(b). Write a C program that uses a recursive function to solve the Towers of Hanoi problem.

Aim: To implement the Towers of Hanoi problem using a recursive function in C programming language.

System and Software Requirements: A Computer with any Operating system with GCC compiler installed in it.

Description:
The Towers of Hanoi is a classic problem that involves moving a stack of disks from one rod to another, with the restriction that a disk can only be placed on top of a larger disk. The problem can be solved using a recursive function in C programming language by breaking down the problem into smaller sub-problems.
The basic idea behind the solution is to move the top n-1 disks from the source rod to the auxiliary rod, then move the nth disk from the source rod to the destination rod, and finally move the n-1 disks from the auxiliary rod to the destination rod.
The Towers of Hanoi is a classic problem that involves moving a stack of disks from one rod to another, with the restriction that a disk can only be placed on top of a larger disk. The problem can be solved using a recursive function in C programming language by breaking down the problem into smaller sub-problems.
Consider a case with 5 disks. The steps to solve the problem are as follows:

  • Move the top 4 disks from the source rod to the auxiliary rod.
  • Move the 5th disk from the source rod to the destination rod.
  • Move the 4 disks from the auxiliary rod to the destination rod.
    • This process can be represented by a recursive function, where each call to the function represents a step in the solution. The function takes four arguments: the number of disks n, the name of the source rod from_rod, the name of the destination rod to_rod, and the name of the auxiliary rod aux_rod.
    The base case of the function is when n is equal to 1, in which case the disk is simply moved from the source rod to the destination rod. In all other cases, the function first calls itself with n-1 as the number of disks, the source rod as from_rod, the auxiliary rod as to_rod, and the destination rod as aux_rod. This moves the top n-1 disks from the source rod to the auxiliary rod.
    Next, the function prints a message indicating that disk n is being moved from the source rod to the destination rod.
    Finally, the function calls itself again with n-1 as the number of disks, the auxiliary rod as from_rod, the destination rod as to_rod, and the source rod as aux_rod. This moves the top n-1 disks from the auxiliary rod to the destination rod.
    This process continues until n is equal to 1, at which point the base case is reached and the function returns, printing the steps to solve the Towers of Hanoi problem.

    #include<stdio.h>
    void towerOfHanoi(int n, char from_rod, char to_rod, char aux_rod)
    {
    if (n == 1)
    {
    printf("Move disk 1 from rod %c to rod %c\n", from_rod, to_rod);
    return;
    }
    towerOfHanoi(n - 1, from_rod, aux_rod, to_rod);
    printf("Move disk %d from rod %c to rod %c\n", n, from_rod, to_rod);
    towerOfHanoi(n - 1, aux_rod, to_rod, from_rod);
    }
    int main()
    {
    int n = 5; // Number of disks
    towerOfHanoi(n, 'A', 'C', 'B'); // A, B and C are names of rods
    return 0;
    }
