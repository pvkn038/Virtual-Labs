Q3(b). Write a C program to calculate the following Sum: Sum=1-x2/2! +x4/4!-x6/6!+x8/8!-x10/10!

Aim: To calculate the following Sum: Sum=1-x2/2! +x4/4!-x6/6!+x8/8!-x10/10!

System and Software Requirements: A Computer with any Operating system with GCC compiler installed in it.

Description:
This is the Taylor Series expansion of the cosine function. The formula represents an infinite series of terms, where each term is calculated by raising x to a power and dividing by the factorial of that power. The terms alternate in sign, with odd powers having a negative sign and even powers having a positive sign. The series approximates the value of the cosine function for a given value of x, with more terms providing a more accurate approximation..
The sum represents the Maclaurin Series expansion of the cosine function. Maclaurin Series is a type of Taylor Series, which is a representation of a function as an infinite sum of terms calculated based on the function's derivatives at a single point, typically at zero. In this particular series, the expansion is around x=0 and the terms alternate in sign, with odd powers having a negative sign and even powers having a positive sign. As the number of terms increases, the sum approaches the actual value of the cosine function for small values of x.
Here the calculation of the sum for x=1:
Sum = 1 - (1^2)/2! + (1^4)/4! - (1^6)/6! + (1^8)/8! - (1^10)/10!
= 1 - 1/2 + 1/24 - 1/720 + 1/40320 - 1/3628800
= 0.540302
The sum cannot be calculated as it is an infinite series. The more terms you add, the more accurate the approximation of the cosine function becomes, but the series itself does not have a finite sum. To obtain a numerical value, you can either approximate the sum using a finite number of terms or use a built-in cosine function in a programming language or calculator.
This program, factorial is a function that calculates the factorial of a number, and the main function calculates the sum using a loop that iterates 10 times, adding each term to the sum. The sign of each term is alternated using the sign variable. The value of x is obtained from the user using the scanf function.

In this example, each term is calculated by raising x to the power indicated in the sum and dividing by the factorial of that power. The result is the sum of these terms, which approaches the actual value of some function for x=1 (the specific function depends on the context).

#include<stdio.h>
#include<math.h>
int factorial(int n) {
int result = 1;
for (int i = 1; i <= n; i++) {
result *= i;
}
return result;
}
int main() {
double x, sum = 1;
int sign = -1;
printf("Enter the value of x: ");
scanf("%lf", &x);
for (int i = 2; i <= 10; i += 2) {
sum += sign * pow(x, i) / factorial(i);
sign *= -1;
}
printf("Sum = %lf\n", sum);
return 0;
}
