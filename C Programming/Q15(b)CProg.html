Q15(b). Define a macro that finds the maximum of two numbers. Write a C program that uses the macro and prints the maximum of two numbers.

Aim: To define a macro that finds the maximum of two numbers, and write a C program that uses the macro and prints the maximum of two numbers.

System and Software Requirements: A Computer with any Operating system with GCC compiler installed in it.

Description:
A macro is a simple text substitution mechanism provided by the preprocessor in C programming language. Macros are expanded in the source code before the actual compilation takes place. Macros can be useful for defining frequently used operations, or for abstracting complex expressions, to make the code more readable and maintainable.
In this case, the macro MAX(a, b) is defined to find the maximum of two numbers, a and b. The macro uses the ternary operator ? : to check which of a and b is larger and return that value. The expression (a) > (b) is evaluated first. If it is true, then a is returned, otherwise b is returned.
Here's how the macro can be used in a program:
#include<stdio.h>
#define MAX(a, b) ((a) > (b) ? (a) : (b))
int main() {
int x = 5, y = 10;
int max = MAX(x, y);
printf("The maximum of %d and %d is %d\n", x, y, max);
return 0;
}
This program defines the macro MAX(a, b), and then uses it to find the maximum of two variables x and y, which are set to 5 and 10, respectively. The result is stored in the variable max, which is then printed to the console.
The output of the program will be:
The maximum of 5 and 10 is 10
Note: Macros can lead to unexpected results and hard-to-debug code if used improperly, so it's important to use them with caution.



Examples:
For Example: Let's use the macro MAX to find and print the maximum of two numbers:

}
In this example, the macro MAX is defined to find the maximum of two numbers a and b. The macro uses the ternary operator ? : to compare the values of a and b, and return the larger of the two.
In the main function, two variables x and y are defined and assigned the values 5 and 10, respectively. The macro MAX is then used to find the maximum of x and y, and the result is stored in the variable max.
Finally, the maximum value is printed to the console using the printf function. The output of the program will be: The maximum of 5 and 10 is 10
This example demonstrates how macros can be used to abstract and simplify frequently used operations in C programming, making the code more readable and maintainable.
