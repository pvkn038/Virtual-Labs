Q2(a). Write a C program to determine if the given number is a prime number or not.

Aim: To determine if the given number is a prime number or not.

System and Software Requirements: A Computer with any Operating system with GCC compiler installed in it.

Description:
One way to determine if a given number is prime is to iterate from 2 to the square root of the number, and check if the number is divisible by any of those numbers. If it is not divisible by any of them, then it is a prime number.
For example, let's say we want to determine if the number 13 is prime. We can start by iterating from 2 to the square root of 13.
We can check if 13 is divisible by 2, 3, 4, and so on until we reach. Since 13 is not divisible by any of those numbers, we can conclude that 13 is a prime number.
Another way is to use the Miller-Rabin primality test, which is a probabilistic test that can quickly determine if a number is likely to be prime.
For example, let's say we want to determine if the number 13 is prime using Miller-Rabin primality test. First, we will choose some random number (a) from 2 to n-2. Then we will calculate (a^d)%n. Note that Miller-Rabin primality test is probabilistic and not deterministic. It can only tell if a number is probably prime, but not if a number is definitely prime.

For Example: Take a number 45. Is it a prime number?
Factors of 45 are 5 x 3 x 3.Since the factors of 45 are more than 2 we can say that 45 is not a prime number but a composite number.

int main()
{
int n, i, flag = 0;
printf("Enter a positive integer: ");
scanf("%d", &n);
for (i = 2; i <= n / 2; ++i)
{
if (n % i == 0)
{
flag = 1;
break;
}
}
if (n == 1)
{
printf("1 is neither a prime nor a composite number.");
}
else
{
if (flag == 0)
printf("%d is a prime number.", n);
else
printf("%d is not a prime number.", n);
}
return 0;
}
