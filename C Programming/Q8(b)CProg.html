Q8(b). Write a C program that reads two matrices and uses functions to perform the following:
i) Addition of two matrices
ii) Multiplication of two matrices

Aim: To design and implement a C program that can perform matrix addition and multiplication by reading in two matrices and using functions.

System and Software Requirements: A Computer with any Operating system with GCC compiler installed in it.

Description:
In this experiment, we will develop a C program that reads two matrices and uses functions to perform matrix addition and multiplication. The program will consist of the following steps:
Input: The program will prompt the user to enter the dimensions of the two matrices (number of rows and columns).
Initialization: The program will allocate memory dynamically to store the values of the two matrices using two-dimensional arrays.
Matrix Input: The program will prompt the user to enter the values of the two matrices.
Function Call: The program will call two functions, one for matrix addition and one for matrix multiplication.
Matrix Addition: The matrix addition function will receive the two matrices as inputs and perform the addition operation. The result will be stored in a new matrix.
Matrix Multiplication: The matrix multiplication function will receive the two matrices as inputs and perform the multiplication operation. The result will be stored in a new matrix.
Output: The program will display the results of the addition and multiplication operations.
Deallocation: The program will free the dynamically allocated memory.
This experiment will help us understand the concepts of dynamic memory allocation and function calls in C programming. It will also help us understand the algorithms for matrix addition and multiplication.

i)Addition of two matrices:
#include<stdio.h>
int main()
{
int m, n, c, d, first[10][10], second[10][10], sum[10][10];
printf("Enter the number of rows and columns of matrix\n");
scanf("%d%d", & m, & n);
printf("Enter the elements of first matrix\n");
for (c = 0; c < m; c++)
for (d = 0; d < n; d++) scanf("%d", & first[c][d]);
printf("Enter the elements of second matrix\n");
for (c = 0; c < m; c++)
for (d = 0; d < n; d++) scanf("%d", & second[c][d]);
printf("Sum of entered matrices:-\n");
for (c = 0; c < m; c++)
{
for (d = 0; d < n; d++)
{
sum[c][d] = first[c][d] + second[c][d];
printf("%d\t", sum[c][d]);
}
printf("\n");
}
return 0;
}

ii) Multiplication of two matrices:
#include<stdio.h>
#include<stdlib.h>
int main(){
int a[10][10],b[10][10],mul[10][10],r,c,i,j,k;
printf("enter the number of row=");
scanf("%d",&r);
printf("enter the number of column=");
scanf("%d",&c);
printf("enter the first matrix element=\n");
for(i=0;i < r;i++)
{
for(j=0;j < c;j++)
{
scanf("%d",&a[i][j]);
}
}
printf("enter the second matrix element=\n");
for(i=0;i < r;i++)
{
for(j=0;j < c;j++)
{
scanf("%d",&b[i][j]);
}
}
printf("multiply of the matrix=\n");
for(i=0;i < r;i++)
{
for(j=0;j < c;j++)
{
mul[i][j]=0;
for(k=0;k < c;k++)
{
mul[i][j]+=a[i][k]*b[k][j];
}
}
}
//for printing result
for(i=0;i < r;i++)
{
for(j=0;j < c;j++)
{
printf("%d\t",mul[i][j]);
}
printf("\n");
}
return 0;
}
