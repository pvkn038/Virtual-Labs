Q7(b). Write a C program that uses non recursive function to search for a Key value in a given sorted list of integers. Use binary search method.

Aim: To implement a non-recursive binary search function in C to efficiently search for a key value in a sorted list of integers.

System and Software Requirements: A Computer with any Operating system with GCC compiler installed in it.

Description:
Binary search is a well-known algorithm for searching a specific key value in a sorted list of elements. The idea behind binary search is to divide the list into two halves at each iteration and eliminate half of the list as a possibility for the key value based on the comparison between the key value and the middle element of the current list.
In a non-recursive binary search, the algorithm uses a loop to perform the same division and comparison process until the key value is found or it can be concluded that the key value is not in the list. To implement a non-recursive binary search function in C, we need to keep track of the starting and ending indices of the current list being considered. At each iteration, we calculate the middle index and compare the middle element with the key value. If the middle element is equal to the key value, we return the middle index. If the middle element is less than the key value, we update the starting index to be one position after the middle index. If the middle element is greater than the key value, we update the ending index to be one position before the middle index. This process continues until the key value is found or the starting index becomes greater than the ending index, in which case the key value is not in the list.
In summary, the non-recursive binary search algorithm in C uses a loop to repeatedly divide the list into two halves, compare the middle element with the key value, and update the starting and ending indices until the key value is found or it can be concluded that it is not in the list.
For example, Consider a sorted list of integers A = [1, 3, 4, 6, 7, 8, 10] and a key value k = 6. The aim is to find the index of k in the list A.
1.Initialize the starting index start to 0 and the ending index end to length of A - 1
start = 0;
end = 6;
2.While start <= end:
a. Calculate the middle index mid as (start + end) / 2
mid = (0 + 6) / 2 = 3
b. Compare A[mid] with k. If they are equal, return mid as the index of k in the list A
A[mid] = 6
6 == 6
return mid = 3
c. If A[mid] < k, update start to mid + 1
A[mid] = 6 < k = 6
start = mid + 1 = 4
d. If A[mid] > k, update end to mid - 1
3.If start > end, the key value k is not in the list A.
In this example, the binary search function returns mid = 3 as the index of k = 6 in the list A.

#include<stdio.h>
int binary_search(int arr[], int n, int key) {
int left = 0, right = n - 1;
while (left <= right) {
int mid = left + (right - left) / 2;
if (arr[mid] == key) {
return mid;
} else if (arr[mid] < key) {
left = mid + 1;
} else {
right = mid - 1;
}
}
return -1;
}
int main() {
int arr[] = {1, 3, 4, 6, 7, 8, 10};
int n = sizeof(arr) / sizeof(arr[0]);
int key = 6;
int index = binary_search(arr, n, key);
if (index != -1) {
printf("Key found at index %d\n", index);
} else {
printf("Key not found\n");
}
return 0;
}
