Q16(b). Write a C Program to calculate the sum of n numbers entered by the user using calloc( ) and free() functions.

Aim: To implement a C Program to calculate the sum of n numbers entered by the user using calloc( ) and free() functions.

System and Software Requirements: A Computer with any Operating system with GCC compiler installed in it.

Description:
The calloc() function can be used in the same way as the malloc() function to dynamically allocate memory for a set of variables. The main difference between malloc() and calloc() is that calloc() initializes the allocated memory block to zero, whereas malloc() does not. This can be useful when you want to ensure that the memory block is initialized to a known value before using it.
Here is a theoretical experiment to calculate the sum of n numbers entered by the user using calloc() and free() functions:
1. Call the calloc() function to allocate memory for n numbers. The calloc() function takes two arguments: the number of elements to be allocated and the size of each element. In this case, the number of elements is n and the size of each element is sizeof(int).
2. Check the return value of the calloc() function to ensure that the memory allocation was successful. If the allocation failed, print an error message and exit the program.
3. Read n numbers from the user and store them in the memory block that was allocated by the calloc() function.
4. Calculate the sum of the n numbers by adding them up one by one.
5. Print the sum of the numbers.
6. Call the free() function to deallocate the memory block that was allocated by the calloc() function. This releases the memory back to the system and prevents memory leaks.
This experiment demonstrates how the calloc() and free() functions can be used to dynamically allocate memory for a set of variables, calculate the sum of the variables, and deallocate the memory after it is no longer needed. By using dynamic memory allocation, you can improve the efficiency of your program and avoid memory leaks.





Examples:
For Example: Here is a example to demonstrate how calloc() and free() functions can be used to calculate the sum of n numbers entered by the user:
1. Allocate memory for n numbers using the calloc() function: p = calloc(n, sizeof(int)). This creates a block of memory that can store n integers.
2. Read n numbers from the user and store them in the memory block that was allocated by the calloc() function: p[0] = num1, p[1] = num2, ..., p[n-1] = numn.
3. Calculate the sum of the n numbers by adding them up one by one: sum = p[0] + p[1] + ... + p[n-1].
4. Print the sum of the numbers: printf("The sum of the numbers is: %d\n", sum).
5. Deallocate the memory block that was allocated by the calloc() function using the free() function: free(p). This releases the memory back to the system and prevents memory leaks.
This simple example demonstrates how the calloc() and free() functions can be used to dynamically allocate memory for an array of variables, calculate the sum of the variables, and deallocate the memory after it is no longer needed. By using dynamic memory allocation, you can improve the efficiency of your program and avoid memory leaks.

#include<stdio.h>
#include<stdlib.h>
int main() {
int n, i, sum = 0;
int *p;
printf("Enter the number of elements: ");
scanf("%d", &n);
p = (int *)calloc(n, sizeof(int));
if (p == NULL) {
printf("Memory allocation failed\n");
return 1;
}
printf("Enter %d numbers: \n", n);
for (i = 0; i < n; i++) {
scanf("%d", &p[i]);
sum += p[i];
}
printf("The sum of the numbers is: %d\n", sum);
free(p);
return 0;
}
