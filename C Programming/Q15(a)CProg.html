Q15(a). Write a C program to merge two files into a third file (i.e., the contents of the first file followed by those of the second are put in the third file).

Aim: To implement a C program to merge two files into a third file (i.e., the contents of the first file followed by those of the second are put in the third file).

System and Software Requirements: A Computer with any Operating system with GCC compiler installed in it.

Description:
The goal of this experiment is to develop a program that can merge two files into a third file in a specified order. The contents of the first file will be followed by the contents of the second file in the third file.
Methodology:
Choose two files as input files.
Read the contents of the first file and store it in a variable.
Read the contents of the second file and store it in another variable.
Create a third file and open it in write mode.
Write the contents of the first file to the third file.
Write the contents of the second file to the third file immediately following the contents of the first file.
Close the third file.
The third file will contain the contents of the first file followed by the contents of the second file, in the same order as they appeared in the original files.
The contents of the third file will be examined to ensure that the merging process was successful and that the order of the contents was preserved. The experiment will also evaluate the performance and efficiency of the program.
The experiment will provide a solution to the problem of merging two files into a third file in a specified order. The results of the experiment will be useful in data management and organization, as well as in other fields where file merging is a necessary task.



Examples:
For Example: Let A and B be two sets of data represented by two files. Each set contains N elements, where N is a positive integer. The elements of set A are denoted by a1, a2, a3, ..., aN, and the elements of set B are denoted by b1, b2, b3, ..., bN.
Read the elements of set A and store them in a list L1.
Read the elements of set B and store them in another list L2.
Create a third list L3, and initialize it with the elements of L1.
Append the elements of L2 to L3.
Write the elements of L3 to a third file, in the same order as they appear in the list.
The third file will contain N elements from set A followed by N elements from set B, in the order a1, a2, a3, ..., aN, b1, b2, b3, ..., bN.
The contents of the third file will be examined to ensure that the merging process was successful and that the elements of set A appear before the elements of set B.
The experiment demonstrates the process of merging two sets of data into a third set of data, preserving the order of the elements in the original sets. This process can be applied to various fields, such as data management and organization, where the merging of multiple sets of data is necessary.

#include<stdio.h>
#include<stdlib.h>
int main() {
FILE *file1, *file2, *mergedFile;
char c;
// Open the first file in read mode
file1 = fopen("file1.txt", "r");
if (file1 == NULL) {
printf("Error opening file1.txt\n");
return 1;
}
// Open the second file in read mode
file2 = fopen("file2.txt", "r");
if (file2 == NULL) {
printf("Error opening file2.txt\n");
fclose(file1);
return 1;
}
// Open the merged file in write mode
mergedFile = fopen("mergedFile.txt", "w");
if (mergedFile == NULL) {
printf("Error creating mergedFile.txt\n");
fclose(file1);
fclose(file2);
return 1;
}
// Write the contents of the first file to the merged file
while ((c = fgetc(file1)) != EOF) {
fputc(c, mergedFile);
}
// Write the contents of the second file to the merged file
while ((c = fgetc(file2)) != EOF) {
fputc(c, mergedFile);
}
// Close all the files
fclose(file1);
fclose(file2);
fclose(mergedFile);
printf("Files merged successfully.\n");
return 0;
}

#include<stdio.h>
#include<stdlib.h>
#define BUFSIZE 1024
int main() {
FILE *file1, *file2, *mergedFile;
char buffer[BUFSIZE];
int bytesRead;
// Open the first file in read mode
file1 = fopen("file1.txt", "r");
if (file1 == NULL) {
printf("Error opening file1.txt\n");
return 1;
}
// Open the second file in read mode
file2 = fopen("file2.txt", "r");
if (file2 == NULL) {
printf("Error opening file2.txt\n");
fclose(file1);
return 1;
}
// Open the merged file in write mode
mergedFile = fopen("mergedFile.txt", "w");
if (mergedFile == NULL) {
printf("Error creating mergedFile.txt\n");
fclose(file1);
fclose(file2);
return 1;
}
// Write the contents of the first file to the merged file
while ((bytesRead = fread(buffer, 1, BUFSIZE, file1)) > 0) {
fwrite(buffer, 1, bytesRead, mergedFile);
}
// Write the contents of the second file to the merged file
while ((bytesRead = fread(buffer, 1, BUFSIZE, file2)) > 0) {
fwrite(buffer, 1, bytesRead, mergedFile);
}
// Close all the files
fclose(file1);
fclose(file2);
fclose(mergedFile);
printf("Files merged successfully.\n");
return 0;
}
